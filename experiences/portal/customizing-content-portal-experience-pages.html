ページコンテンツ

    ポータル・エクスペリエンスページのコンテンツのカスタマイズ

    このトピックでは、ポータル・エクスペリエンスページのコンテンツをカスタマイズする方法について説明します。

    テキスト、画像、リンクなどのオブジェクトを Portal Experience ページに追加して、動画体験をカスタマイズし、ブランド化することができます。

    注：ページにオブジェクトを追加するプロセスは、テンプレートによって多少異なります。このトピックのスクリーンショットは、マーキーテンプレートを使用して撮影したものです。
    注：一部のページはプレーヤーのレイアウトを変更する機能も備えています。詳細については、「ポータルエクスペリエンスのプレーヤーレイアウトの変更」を参照してください。

    ページへのオブジェクトの追加

    サイトエディタを使用して、画像、テキスト、リンクなどのオブジェクトをページに追加できます。エクスペリエンスの編集から始めて、左側のナビゲーションで [ PAGESS ] をクリックします。ページテンプレートには、ページに追加できるオブジェクトを示すプラスアイコン ( ) が表示されます。プラスアイコンの上にカーソルを置き、クリックして適切なオブジェクトを追加します。

    ページテンプレートには、テンプレートの一部であるテキストオブジェクトを含めることもできます。これらの上にマウスポインタを置くと、 [編集 ] ボタンが表示されます。オブジェクトまたは [ 編集 ] ボタンをクリックして、オブジェクトを編集します。

    追加または編集できるオブジェクトは、エクスペリエンスの初回作成時に選択されたテンプレートに基づきます。

    ページにテキストを追加する

    ページにテキストを追加する方法は、いくつかあります。

    • [ +画像またはテキストを追加 ] リンクをクリックする
    • ページ上のテキストプレースホルダのいずれかをクリックすることにより

    [ +画像またはテキストを追加] リンクをクリックしてテキストを追加します

    [ + イメージを追加] または [テキストの追加] リンクをクリックして、ページにテキストを追加するには :

    1. をクリックします。
    2. テキストを入力します。書式設定バーを使用して、テキストの書式を設定できます。終了後、[Save]をクリックします。

    テキストプレースホルダーをクリックしてテキストを追加する

    ページには、テンプレートの一部として、多数のテキストオブジェクトを含めることができます。テキストオブジェクトをクリックして編集します。

    テキストを追加するには、次の手順を行います。

    1. プレースホルダーオブジェクトの1つをクリックします。
    2. テキストを入力して[Save]をクリックします。

    ページヘッダーに画像を追加する

    画像は、テンプレートに基づいてサイズが変更されます（拡大されることはありません）。画像アップローダーは、画像が使用されている場所に応じて、推奨画像サイズが表示される場合があります。JPEG、PNG、GIF 画像がサポートされています。ページヘッダーに画像を追加するには、次の手順を行います。

    1. をクリックします。
    2. [ イメージ ] タブをクリックします。画像ブラウザダイアログが開きます。
    3. [ Browse ] をクリックし、ファイルシステムからイメージを参照して選択します。終了後、[Save]をクリックします。

    ページヘッダーにナビゲーションリンクを追加する

    ナビゲーションリンクをページ上部に追加し、ユーザーがサイトの他の部分へ移動する方法を提供できます。ページにナビゲーションリンクを追加するには、次の手順を行います。

    1. ページヘッダーの [ +メニューリンクを追加] をクリックします。
    2. テキストとリンクを入力します。リンクを新しいブラウザウィンドウで開きたい場合は、[New Window]を選択します。終了後、[Save]をクリックします。

    追加のナビゲーションリンクを入力するには、[+ADD LINK]をクリックします。リンクを編集するには、リンクをクリックします。

    ページフッターにナビゲーションリンクを追加する

    ナビゲーションリンクは、ページフッターにも追加できます。タイトルの下に一連のリンクを表示したいときに、これは便利です。例は次のとおりです。

    ページフッターナビゲーションオブジェクトは、タイトルの PLAYDER と +Add Menu Links に関連付けられたリンクの 2 つの部分で構成されています。

    ページフッターにナビゲーションリンクを追加するには:

    1. [ プレースホルダ ] をクリックし、ヘッダーの値を入力して [ 保存 ] をクリックします。
    2. [ +メニューリンクを追加] をクリックし、テキスト、リンク URL、およびリストの順序を入力します。リンクの順序を指定することもできます。それには、リンクの作成時[List Order]値を設定します。

    [ + メニューリンクを追加] をクリックすると、追加のナビゲーションリンクを入力できます。

    ページからオブジェクトを削除する

    テキスト、画像、リンクをページから削除する方法は、いくつかあります。

    ページヘッダーからテキストまたは画像を削除する

    1. オブジェクトにカーソルを合わせ、削除アイコンをクリックします。

    ページヘッダーからナビゲーションリンクを削除する

    1. ナビゲーションヘッダーのリンクをクリックします。
    2. [削除]をクリックします。

    ページフッターからナビゲーションリンクを削除する

    1. ページフッターのリンクをクリックします。
    2. [削除]をクリックします。

    ページフッターからタイトルを削除する

    1. ページフッターのタイトルをクリックします。
    2. タイトルテキストを消去して[Save]をクリックします。
