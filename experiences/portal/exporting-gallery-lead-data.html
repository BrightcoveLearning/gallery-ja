ギャラリーリードデータのエクスポート

{{ page.description }}

ギャラリーリードフォームデータは、ギャラリーエクスペリエンスがリードフォームを使用するように構成されている場合に収集されます。潜在顧客データを収集するためのポータルエクスペリエンスの構成については、「ポータルエクスペリエンスでリードフォームを使用する」を参照してください。

オーディエンスモジュールからギャラリーリードをエクスポートする場合、オーディエンスリードフォームを使用してキャプチャされたデータのみがエクスポートされます。カスタムリードフォームからのリードデータは、マーケティング自動化ソフトウェアに保存されます。

注:管理者権限を持つユーザーだけが、リードデータをダウンロードできます。

ギャラリーのリードデータをエクスポートするには、次の手順に従います。

  1. オーディエンスモジュールを開きます。
  2. クリックギャラリーリードのエクスポート左側のナビゲーションで。
  3. リードをエクスポートするギャラリーエクスペリエンスを選択します。
  4. リードの「開始日」と「終了日」を入力します。
  5. エクスポート形式、カンマ区切り値、または Excel を選択します。
  6. [ エクスポート]ボタンをクリックして、リードデータをダウンロードします。

エクスポートされたファイルには、次のフィールドが含まれます。

  • 可能なすべてのリードフォームフィールド
    • emailAddress
    • 名
    • lastName
    • BusinessPhone
    • 国
    • 会社名
    • 業界
  • externalID -ビューアを一意に識別するために使用される ID
  • PlayerID -リードフォームを表示したプレイヤー ID
  • VideoID -データが収集されたときに視聴されるビデオのビデオ ID
  • pageURL -データが収集されたときにページ URL ビューアがオンになっていた
  • CreateDat -データが収集された日付と時刻
  • エクスペリエンス -データが収集されたときに表示されるエクスペリエンスの名前
