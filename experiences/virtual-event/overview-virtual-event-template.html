仮想イベントテンプレートの概要

{{ page.description }}

ヘッダー

ヘッダーはすべてのページの上部に表示されます。ヘッダーは、設定ファイルの値から構築されます。ヘッダーは、次のコンポーネントで構成されています。

  • ヘッダーロゴ
  • リンク集
  • 検索

ヘッダーのカスタマイズについては、仮想イベントエクスペリエンスページのコンテンツのカスタマイズを参照してください。

ようこそメッセージ

ウェルカムメッセージは、サイトへの訪問者を歓迎するために使用でき、テキストと画像を含めることができます。カスタム HTML は、より高度なメッセージに使用できます。この例のウェルカムメッセージは、テキストと画像の組み合わせを使用して作成されました。

ウェルカムメッセージの追加については、仮想イベントエクスペリエンスページのコンテンツのカスタマイズを参照してください。

注目のコレクション

注目のコレクションは、テンプレートに表示される最初のビデオコレクションです。注目のコレクションは、通常、ライブビデオを紹介するために使用されます。ナビゲーション矢印を使用すると、ユーザーは注目の動画内を移動できます。Q&A ウィジェットを各動画に関連付けることもできます。コレクションの作成とビデオの追加については、バーチャルイベントエクスペリエンスへのビデオの追加を参照してください。

注:注目のコレクションでは、最大 24 本の動画がサポートされます。

プライマリコレクション

プライマリコレクションは、テンプレートに表示される 2 番目のビデオコレクションです。ナビゲーション矢印を使用すると、ユーザーはビデオ内を移動できます。コレクションの作成とビデオの追加については、バーチャルイベントエクスペリエンスへのビデオの追加を参照してください。

注:Primary コレクションでは、最大 24 本のビデオがサポートされます。

議題

アジェンダは通常、ライブイベントのスケジュールを一覧表示するために使用されます。アジェンダには、セッション名、スピーカー、およびセッションの日時が表示されます。セッションの横にカウントダウンタイマーが表示され、セッションがライブになると [表示] ボタンが表示されます。イベントアジェンダの作成については、「イベントアジェンダの管理」を参照してください。

スピーカー

スピーカーセクションには、すべてのイベントスピーカーが自動的に入力されます。すべてのセッションスピーカーが[スピーカー]セクションに表示されます。スピーカーセクションには、次のものが表示されます。

  • スピーカー画像
  • 名前
  • タイトル
  • 会社
  • 短い略歴

イベントスピーカーの管理については、イベントスピーカーの管理を参照してください。

二次的コレクション

セカンダリコレクションは、テンプレートに表示される 3 番目のビデオコレクションです。ナビゲーション矢印を使用すると、ユーザーはビデオ内を移動できます。コレクションの作成とビデオの追加については、バーチャルイベントエクスペリエンスへのビデオの追加を参照してください。

注:セカンダリコレクションでは、最大 24 本のビデオがサポートされます。

[トラック]

トラックエリアは、通常 VOD ビデオを表示するために使用される一連のコレクションです。トラックをクリックすると、そのトラック内のビデオが表示されます。コレクションの作成とビデオの追加については、バーチャルイベントエクスペリエンスへのビデオの追加を参照してください。

静的コンテンツ

詳細グリッドは、設定ファイル内の静的テキストオブジェクトを使用して作成されます。静的テキストオブジェクトは、カスタムHTMLをテンプレートに挿入するために使用されます。

静的コンテンツは任意の HTML であり、テンプレートの任意の場所に挿入できます。静的コンテンツオブジェクトの操作については、静的コンテンツの操作を参照してください。

スポンサー

スポンサーセクションを使用して、イベントのスポンサーを強調することができます。スポンサーごとに、次の項目を表示できます。

  • スポンサーロゴ
  • スポンサーのウェブページへのリンク

スポンサーセクションのカスタマイズについては、イベントスポンサーのカスタマイズを参照してください。

フッター

フッターはすべてのページの下部に表示されます。フッターは、設定ファイルの値から作成されます。フッターは、次のコンポーネントで構成されています。

  • 主吹き出し領域
  • 法的テキストと著作権
  • リンクの 3 列
  • ソーシャルアイコン
