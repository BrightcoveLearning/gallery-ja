仮想イベントテンプレートの仕様

{{ page.description }}

動画と音声

ビデオとオーディオの推奨事項を以下に示します。

ビデオ

フォーマット MP4
コーデック H.264、H.265
ビットレート 最高のレンディションの 2 倍のビットレート
フレームレート 定数。通常は 15 ～ 60 フレーム/秒
アスペクト比 動画のアスペクト比は保持されます。16:9 のアスペクト比が一般的です。

オーディオ

コーデック AAC（高度なオーディオコーデック）
ビットレート 定数、128kbps
チャンネル ステレオ

最適な再生を実現するための動画の録画、エクスポート、アップロード、公開に関するベストプラクティスに関する推奨事項については、ベストプラクティスを参照してください。動画の作成、トランスコーディング、アップロードのワークフロー。

画像

仮想イベントテンプレート全体では、さまざまなイメージが使用されます。画像の推奨事項を以下に示します。

背景画像

イメージのサイズは、お客様がサポートしようとしているブレークポイントによって異なります。1920x1080では動作しない場合が多いのですが、最適なパフォーマンスを得るには大きくならないようにイメージを最適化する必要があります。

ヘッダーロゴ

ヘッダーロゴ画像は、縦横に 178px に縮小されます。

議題

デフォルトの画像サイズは320ピクセル×180ピクセルです。画像はこのサイズに切り取られます。

スピーカー

スピーカー画像の最大幅は200pxにする必要があります。画像のサイズは変更されません。デフォルトでは、テンプレート CSS はスピーカーイメージを 50% の境界半径で設定します。これにより、画像が円形になります。スピーカーの下に表示されるスピーカーの略歴は、15語の約3行に制限されています。

スポンサー

スポンサー画像は幅100ピクセルに変更され、不透明度は 80% に設定されます。この動作をオーバーライドするサンプル CSSが提供されています。

フォント

テンプレートは、 Roboto Consedフォントで動作するように最適化されています。このフォントは、スタイルギャラリーインターフェース内のメニュー。他のフォントも使用できますが、CSS ファイルの変更が必要になる場合があります。カスタム・フォントの使用の詳細は、「ポータル・エクスペリエンスでのカスタム・フォントの使用」を参照してください。

議題

イベントアジェンダには、ライブイベントやその他のセッションのスケジュールが記載されています。アジェンダが作成されて公開されると、イベントのホームページに表示されます。イベントアジェンダの作成については、「イベントアジェンダの管理」を参照してください。
