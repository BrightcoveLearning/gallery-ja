仮想イベントエクスペリエンスのヘッダーのカスタマイズ

このトピックでは、仮想イベントエクスペリエンスページのヘッダーをカスタマイズする方法について説明します。

ヘッダーはすべてのページの上部に表示されます。ヘッダーは、設定ファイルの値から構築されます。ヘッダーは、次のコンポーネントで構成されています。

  • ヘッダーロゴ
  • リンク集
  • ソーシャルアイコン
  • 検索

ヘッダーロゴ

ヘッダーイメージは、images設定ファイル内のオブジェクトによって指定されます。

  • header_logo -ヘッダーに表示する画像への URL

.Png ファイルと.jpg ファイルがサポートされています。ヘッダーのロゴは、縦横に 178px に縮小されます。

リンク集

デフォルトでは、「ホーム」、「ライブストリーム」、および「オンデマンドセッション」のリンクがテンプレートにハードコードされています。

  • HOME -ホームへ、ページのトップへ
  • ライブストリーム -注目のコレクションに移動します（トップビデオプレーヤー）
  • オンデマンドセッション -トラックセクションに移動します。

header_links設定ファイルでオブジェクトを編集すると、リンクを追加できます。

  • link_url -宛先 URL
  • label -ヘッダーに表示されるラベル

ソーシャルアイコン

デフォルトでは、ヘッダーにはソーシャルアイコンが表示されます。

  • Facebook
  • Instagram
  • Twitter
  • LinkedIn
  • YouTube
ヘッダーに表示されるソーシャルアイコンは、social設定ファイル内のオブジェクトを編集することで編集/削除できます。詳細については、「仮想イベントエクスペリエンスのソーシャルアイコンの設定」を参照してください。

検索

検索を有効にして検索フィールドを表示するには、エクスペリエンスの検索を有効にします。デフォルトでは、ヘッダーの検索フィールドには、テキスト入力用の 1 行が表示されます。

この例の検索ボックスには、検索フィールドの周囲にボックスが表示されるようにスタイルが設定されています。

ヘッダー固有のスタイル設定

モバイルデバイス上のリンク

デフォルトでは、ヘッダーに追加されたリンクは、モバイルデバイスを使用しているときに表示されません。次の JavaScript でこの問題を解決します。


for (let i = 0, l = ciabConfig.header_links.length; i < l; i++) {
  let link = ciabConfig.header_links[i];
  let li = document.createElement('li');
  let ul = document.getElementById('main-nav-cat');
  li.className = 'main-nav-category main-nav-category-home';
  li.innerHTML = ('' + link.label + ''); ul.appendChild(li);
}

検索フィールドの書式設定
