イベントスピーカーのカスタマイズ

このトピックでは、バーチャルイベントエクスペリエンスのホームページに表示されるイベントスピーカーをカスタマイズする方法について説明します。

バーチャルイベントのホームページには、スピーカ画像と略歴を追加できます。

スピーカー情報は、speakers設定ファイル内のオブジェクトを編集してカスタマイズします。

  • name -スピーカー名
  • description -画像の下に表示されるスピーカー情報。これは、15語の約3行に制限されています。
  • image_url -スピーカーのイメージ。最大幅200ピクセルをお勧めします。画像のサイズは変更されません
  • link_url -（オプション）画像の宛先 URL（スピーカーの LinkedIn ページへのリンクなど）

デフォルトでは、テンプレート CSS はスピーカーイメージを 50% に設定し、画像を円形にします。border-radius

speakers設定ファイル内のオブジェクトには、ホームページのどこにスピーカー情報が表示されるかを制御するパラメータもあります。スピーカーセクションの位置付けについては、「ホームページでのコンテンツの配置」を参照してください。
