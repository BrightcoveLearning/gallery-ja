イベントアジェンダの構成

このトピックでは、仮想イベントエクスペリエンスのイベントアジェンダをカスタマイズする方法を学習します。

イベントの議題は、通常、特集コレクションに表示されるスケジュールされたライブイベントを表示するために使用されます。各セッションで表示されるボタンは、セッションの状態によって異なります。

  • イベントの前 -ボタンが表示されず、カウントダウンタイマーが表示されます。
  • 開催中 -イベントが進行中のときに今すぐライブする
  • イベント後 -ボタンは、過去のイベントのセッションを表示します。

議題の日数に基づいて、[ 表示]ドロップダウンが表示され、閲覧者は表示する日のスケジュールを選択できます。

議題情報は、schedule設定ファイル内のオブジェクトを編集することでカスタマイズされます。

  • name -セッション名
  • description -セッションの説明-現在ページには表示されませんが、カスタム JavaScript を使用して表示可能
  • starts_at -セッション開始日時
  • ends_at -セッション終了日時
  • live_id -未使用
  • vod_id -ビデオ ID
  • speakers -セッションスピーカー情報
  • image_url -セッションイメージ。デフォルトの画像サイズは320ピクセル×180ピクセルです。画像はこのサイズに切り取られます。

scheduleコンフィギュレーションファイル内のオブジェクトには、titleスケジュールの上に表示されるのパラメータと、スケジュールを表示するかどうかを設定するためのパラメータもあります。expandedモードです。
