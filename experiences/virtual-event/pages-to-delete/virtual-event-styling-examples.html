仮想イベントスタイリングの例

このトピックでは、仮想イベントテンプレートをカスタマイズするために使用できるサンプルコードをいくつか紹介します。

ソーシャル共有アイコン

デフォルトでは、エクスペリエンスの動画共有設定が有効になっている場合、動画の詳細ページの動画の横に表示されません。次の CSS では、動画の詳細ページにソーシャルアイコンが表示されます。


.share-list {
  display: block !important;
}
.share-list a {
  display: inline-block !important;
}
.share-list li {
  display: block !important;
}
