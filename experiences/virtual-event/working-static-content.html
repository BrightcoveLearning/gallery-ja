静的コンテンツの操作

{{ page.description }}

静的コンテンツオブジェクトは、カスタムHTMLをテンプレートに挿入するために使用されます。このサンプルヒーローバナーは、テキストと画像の組み合わせを使用して構築され、ページの上部に注入されました。

静的コンテンツは任意の HTML であり、テンプレートの任意の場所に挿入できます。静的コンテンツオブジェクトは、static_content設定ファイルのオブジェクトに追加されます。静的コンテンツをページに配置する方法については、「ホームページでのコンテンツの配置」を参照してください。
