サイトエディタを使用して仮想イベントエクスペリエンスをカスタマイズする

{{ page.description }}

Gallery Site Editorは、エクスペリエンスのルックアンドフィールをカスタマイズするために使用されます。エクスペリエンスのタイトルをクリックして、サイトエディタを開きます。サイトエディタには、次の機能があります。

  • ページへのオブジェクトの追加
  • サイトに動画を追加する
  • サイトスタイルをカスタマイズする
  • コール・トゥ・アクションを構成する
  • 外観と動作をカスタマイズする
  • サイト機能のカスタマイズ
  • ビデオと再生をカスタマイズする
  • サイト構成をカスタマイズする
  • サイトの詳細を構成する

これらのオプションにアクセスするには、左側のナビゲーションのメニューをクリックします。

ページにオブジェクトを追加する

テキスト、画像、リンクなどのオブジェクトをページに追加して、動画体験をカスタマイズし、ブランド化することができます。ページにオブジェクトを追加する方法については、「仮想イベントエクスペリエンスページのコンテンツのカスタマイズ」を参照してください。

サイトへのビデオの追加

サイトにビデオを追加するには、をクリックしますビデオ左側のナビゲーションで。ビデオサイトにビデオを追加する方法の詳細については、「バーチャルイベントエクスペリエンスへのビデオの追加」を参照してください。

サイトスタイルのカスタマイズ

サイトで使用されている色やフォントをカスタマイズしたり、別のテーマを選択したりするには、をクリックしますスタイル左側のナビゲーションで。スタイルのカスタマイズ方法の詳細については、「ポータルエクスペリエンスのスタイルのカスタマイズ」を参照してください。

アクションへの呼び出しの設定

アクションへの呼び出しは、ページ上にカスタム HTML、画像、広告を含める機能を提供します。すべてのテンプレートが Call to Action をサポートしているわけではありません。召喚状を設定するには、召喚状左側のナビゲーションで。

アクション呼び出しの構成の詳細については、「ポータル・エクスペリエンスでのアクション・コールの構成」を参照してください。

外観と動作のカスタマイズ

外観と動作の設定は、カスタムヘッダー/フッターの設定、カスタム CSS および JavaScript ファイルの使用、背景画像またはビデオの構成に使用されます。外観と動作の設定を構成するには、をクリックします外観と動作左側のナビゲーションで。

外観と動作の設定の詳細については、「仮想イベントエクスペリエンスの外観と動作をカスタマイズする」を参照してください。

サイト機能のカスタマイズ

サイトの機能設定は、サイト検索の有効化、ソーシャル共有オプションの構成、ビデオへのコメントの許可に使用されます。サイトの機能を構成するには、をクリックしますサイトの機能左側のナビゲーションで。

サイト機能をカスタマイズする方法の詳細については、「ポータルエクスペリエンスのサイト機能のカスタマイズ」を参照してください。

ビデオと再生の設定をカスタマイズする

ビデオと再生の設定では、エクスペリエンスのカスタムプレーヤーの選択、オーディエンスの統合とリードフォームの設定、ダウンロードや関連リンクなどのビデオオプションの構成を行うことができます。ビデオと再生の設定をカスタマイズするには、をクリックしますビデオと再生左側のナビゲーションで。

ビデオおよび再生設定の構成の詳細については、「ポータルエクスペリエンスのビデオおよび再生設定のカスタマイズ」を参照してください。

サイト構成のカスタマイズ

サイト構成設定では、カスタムドメインでサイトをホストしたり、SSL を使用してセキュリティで保護したり、サイトへのアクセスを制限したりできます。サイト構成を構成するには、をクリックしますサイト構成左側のナビゲーションで。

サイト構成設定の詳細については、「ポータルエクスペリエンスのサイト構成のカスタマイズ」を参照してください。

サイト詳細の構成

概要設定では、サイトの名前、説明、言語を編集できます。サイトの詳細を構成するには、左側のナビゲーションの [ 概要 ] をクリックします。

サイト詳細の構成の詳細については、「ポータル・エクスペリエンスの詳細の構成」を参照してください。
