アジェンダモジュール入門

{{ page.description }}

アジェンダモジュールにアクセスするには、仮想イベントエクスペリエンスをクリックしてギャラリーサイトエディタを開き、[ サイトの機能] > [議題] の順にクリックします。

ホームページから以下を行うことができます。

  • イベントの議題をアップロードする
  • イベントの議題を管理する
  • イベントスピーカーを管理する
アジェンダモジュールからギャラリーサイトエディタに戻るには、ページの左側にある [ マイイベントに戻る]リンクをクリックします。

イベントアジェンダのアップロード

イベントアジェンダを最初に作成するには、アジェンダ.csvファイルをアップロードする必要があります。イベント議題のアップロードについては、「議題ファイルのアップロード」を参照してください。

イベントアジェンダの管理

イベントの議題を編集するには、それをクリックして、必要に応じて変更を加えます。詳細については、「イベントアジェンダの管理」を参照してください。

イベントスピーカーの管理

イベントに関連付けられているスピーカーを管理するには、をクリックしますスピーカー左側のナビゲーションで。スピーカーの管理については、イベントスピーカーの管理を参照してください。
