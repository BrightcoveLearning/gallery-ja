イベントスポンサーのカスタマイズ

{{ page.description }}

スポンサー画像は、バーチャルイベントのホームページに追加することができます。

スポンサー情報は、sponsorsコンフィギュレーションファイル内のオブジェクトを編集することによってカスタマイズされます。

  • name -スポンサー名
  • image_url -スポンサーロゴ、画像は幅100ピクセルにサイズ変更されます
  • link_url -画像の宛先 URL

sponsors設定ファイル内のオブジェクトには、ホームページ内のスポンサー情報の表示場所を制御するパラメータもあります。スポンサーセクションの位置付けについては、「ホームページでのコンテンツの配置」を参照してください。

不透明度とサイズ変更ロゴ画像の削除

デフォルトでは、ロゴ画像は 80% の不透明度でスタイル設定されます。次のCSSスタイルを使用して、不透明度を 100% に設定することができます。max-widthプロパティを使用すると、イメージのサイズを変更できます。


<style>
.bc-sponsors-section .bc-sponsors-list img {
    -webkit-filter: none;
    opacity: 1;
    max-width: 200;
}
</style>
エクスペリエンスを編集し、「カスタムヘッダー HTML .」フィールドの上部に CSS を追加します。
