イベントスピーカーの管理

{{ page.description }}

バーチャルイベントのホームページには、スピーカ画像と略歴を追加できます。

イベントスピーカーは、アジェンダモジュールを使用して管理されます。イベントアジェンダファイルのアップロードに含まれていたすべてのイベントスピーカーは、[ イベントスピーカー]リストに表示されます。

注:ホームページには最大8人のスピーカーを表示できます。
注:最大4台のスピーカーを連続して表示することができます。

アジェンダモジュールにアクセスするには、次の手順に従います。

  1. 仮想イベントエクスペリエンスをクリックして、ギャラリーサイトエディターを開きます。
  2. サイト機能 > アジェンダをクリックします。
  3. 左側のナビゲーションで、[ スピーカー] をクリックします。デフォルトでは、[ イベントスピーカー]リンクが選択されており、現在のイベントのすべてのスピーカーが表示されます。

すべてのスピーカーリンクをクリックすると、すべての仮想イベントエクスペリエンスに割り当てられているすべてのスピーカーが表示されます。スピーカーリストは、列ヘッダー名をクリックして並べ替えることができます。

スピーカーの作成

新しいスピーカーを作成するには、次の手順に従います。

  1. [ スピーカーを作成] をクリックします。
  2. スピーカーを入力してくださいファーストネームと苗字。
  3. スピーカーの E メールを入力します。
  4. スピーカータイトルと会社名を入力します。
  5. スピーカーの Bio を入力します。
  6. ヘッドショット画像の場合は、[ 画像のアップロード]または [アップロード] アイコンをクリックし、スピーカー画像を参照して選択します。
  7. [保存]をクリックします。

スピーカーの編集

新しいスピーカーを編集するには、次の手順に従います。

  1. スピーカーをクリックして、[ スピーカーの編集]パネルを開きます。
  2. 必要な変更を加えます。
  3. [保存]をクリックします。

スピーカーの削除

スピーカーを削除するには、次の手順に従います。

  1. スピーカーをクリックして、[ スピーカーの編集]パネルを開きます。
  2. [ スピーカーを削除] をクリックします。
  3. [ スピーカーの削除] をクリックして、削除を確定します。

スピーカーを探す

NS スピーカーを検索ページ上部のフィールドを使用して、スピーカーを検索できます。[ イベントスピーカー]または [ すべてのスピーカー]リンクをクリックして、検索するスピーカーを指定します。話者の名前、名前、役職、会社のフィールドが検索されます。
