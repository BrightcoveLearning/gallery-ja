イベントアジェンダの管理

{{ page.description }}

アジェンダモジュールは、外部の.csvアジェンダファイルのアップロードをサポートしているため、すべてのアジェンダアイテムを手動でアジェンダモジュールに入力する必要はありません。アジェンダファイルのアップロードは、イベントアジェンダが最初に作成される方法です。アジェンダをアップロードした後、アジェンダモジュールを使用して編集できます。

注:アップロード用の議題ファイルの作成については、「イベント議題ファイルの作成」を参照してください。

アジェンダファイルのアップロード

既存のアジェンダをアジェンダモジュールにインポートする前に、アジェンダファイルを.csvファイルにエクスポートする必要があります。アジェンダモジュールは、.csvファイルのみをインポートできます。

Gallery Site Editorを使用して、イベントアジェンダをアップロードおよび構成できます。アジェンダファイルを仮想イベントエクスペリエンスにアップロードするには、次の手順に従います。

  1. エクスペリエンス名をクリックして、ギャラリーサイトエディターを開きます。
  2. 左側のナビゲーションで、[ サイトの機能] > [議題] をクリックします。
  3. アジェンダのアップロード] をクリックします。
  4. アップロードするアジェンダファイルを参照して選択します。
  5. アジェンダが [ マイアジェンダ]ページにリストされていることを確認します。

アジェンダのインポート中にエラーが発生した場合は、メッセージが表示されます。インポート後、ステータスは非アクティブ化されます。ステータスがDEACTIVATEDのアジェンダは、仮想イベントエクスペリエンスに表示されません。

アジェンダタイトルの編集

アジェンダのタイトルは、アジェンダの上にあるエクスペリエンスホームページに表示されます。デフォルトでは、アジェンダタイトルはインポートされたアジェンダファイル名に設定されます。

議題のタイトルを編集するには、編集アイコンをクリックし、タイトルを入力して Enter キーを押します。

議題を表示する

アジェンダファイルがアップロードされると、[ マイアジェンダ ] ページに表示されます。議題の詳細を表示するには、イベント名をクリックします。議題が開き、詳細が表示されます。

ページの左側にある日付をクリックして、その日の議題を表示します。トラックは、ページ上部の列ヘッダーに表示されます。

セッションの追加

新しいセッションを追加するには、次の手順に従います。

  1. アジェンダをクリックして、アジェンダエディタを開きます。
  2. 空き時間枠をクリックします。
  3. セッション情報を入力します。
    • セッション名
    • スピーカー-クリックスピーカーを選択と1つ以上のスピーカーを選択しますセッションのために
    • 開始日
    • 開始時刻
    • 長さ
    • 説明
    • トラック-セッションを割り当てるトラックを選択します
    • セッションタイプ
      • セッション
      • 基調
      • ミーティング
      • その他
    • ビデオID-セッション、基調講演、その他のセッションタイプで利用可能
    • URL-会議セッションタイプで使用可能
    • 表紙画像
    • 関連コンテンツ-セッションコンテンツ（配布物またはスライド）へのリンクを提供するために使用できます
      • ラベル-コンテンツのラベル
      • URL-コンテンツへのリンク
  4. [保存]をクリックします。

セッションがアジェンダに追加されていることを確認します。同じトラックに同時にスケジュールされた複数のセッションがある場合、それらは強調表示され、隣り合って表示されます。

セッションの編集と削除

セッションを編集または削除するには、セッションをクリックして[セッションの編集]パネルを開きます。セッションを編集するには、変更を行い、[ 保存] をクリックします。セッションを削除するには、[ セッションの削除] をクリックします。

セッションへのスピーカーの追加

セッションにスピーカーを追加するには、次の手順に従います。

  1. セッションをクリックして、セッションエディタを開きます。
  2. [ スピーカーを選択] をクリックします。
  3. 1 つ以上のスピーカーを選択するか、[ スピーカーの作成] をクリックして新しいスピーカーを作成します。
    注:スピーカーの作成については、イベントスピーカーの管理を参照してください。
  4. [保存]をクリックします。
[ 検索 ] フィールドは、スピーカーを検索するために使用できます。

新しいトラックを追加する

トラックは、関連するセッションをグループ化するために使用されます。たとえば、マーケティングトラックには、すべてのマーケティング関連セッションが含まれます。アジェンダには通常、複数のトラックがあります。新しいトラックを追加するには、次の手順に従います。

  1. アジェンダをクリックして、アジェンダエディタを開きます。
  2. アジェンダの上部にある [ 新しいトラックを追加]リンクをクリックします。リンクを表示するには、アジェンダを右にスクロールする必要がある場合があります。
  3. トラック名を入力し、 Enter キーを押します。

トラックの編集

トラックを編集するには、トラック名をクリックして変更を加え、 Enterを押して変更を保存します。

トラックの並べ替え

アジェンダ内のトラックの順序を変更するには、次の手順に従います。

  1. アジェンダをクリックして、アジェンダエディタを開きます。
  2. トラック名にカーソルを合わせて移動し、グラブアイコンをクリックして、トラックを新しい場所にドラッグします。

アジェンダのアクティブ化と非アクティブ化

アジェンダステータスは、アジェンダが関連する仮想イベントエクスペリエンスに表示されるかどうかを決定します。ステータスは、アジェンダ名の下のページヘッダーに表示されます。

  • 活性化 -アジェンダは、関連する仮想イベントエクスペリエンスに表示されます
  • 非アクティブ化 -アジェンダは、関連する仮想イベントエクスペリエンスには表示されません

アジェンダをアクティブにするには、をクリックします活性化ページの上部にあります。アジェンダを無効にするには、アジェンダをクリックし、[ 非アクティブ化] をクリックします。アジェンダをアクティブ化/非アクティブ化した後、エクスペリエンスを再公開する必要がないことに注意してください。
