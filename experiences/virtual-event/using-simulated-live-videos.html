シミュレートされたライブビデオの使用シミュレートされたライブビデオの使用方法

{{ page.description }}

Simuliveはの略です シミュレートされたライブ これは、VODコンテンツをライブであるかのようにエンドユーザーに表示できる方法です。Simulive コンテンツは、コンテンツの事前録音とライブ前の広範なテストを可能にするため、イベント主催者にとって魅力的です。エンコーダの問題やその他のロジスティクスによって発生する従来のライブイベントにおいて、障害発生の可能性が軽減されます。Simulive 動画には、ライブになるタイミングを示すカウントダウンタイマーがビデオとともに表示されます。以下の例は、注目のコレクションの simulive ビデオです。

プライマリコレクションとセカンダリコレクションのビデオの場合、カウントダウンタイマーがビデオのカードに表示されます。

バーチャルイベントエクスペリエンスで simulive コンテンツを使用するには、次の操作を行う必要があります。

  • Simulive Player プラグインを設定する
  • プレイヤーをエクスペリエンスに関連付ける
  • Simulive カスタムフィールドを作成する
  • Simuliveビデオプロパティの設定
注:カウントダウンタイマーの一部として表示されるテキストは、labels設定ファイルのオブジェクト内にあり、簡単にカスタマイズまたはローカライズできます。

Simulive Player プラグインの設定

Simulive プレーヤープラグインは、バーチャルイベントエクスペリエンスで使用するプレーヤー用に構成する必要があります。プラグインの構成に必要な情報を入手するには、アカウントマネージャーに連絡してください。

  1. ナビゲーションヘッダーの[Players]をクリックし、Playersモジュールを開きます。
  2. ブライトコーブでは、仮想イベントエクスペリエンスのための専用プレーヤーを作成することをお勧めします。専用プレイヤーを作成していない場合は、[ 新規プレイヤー] をクリックし、プレイヤーの名前を付け、[ 保存] をクリックします。
  3. 設定するプレーヤーのリンクをクリックします。
  4. 左側のナビゲーションメニューで [ プラグイン ] をクリックします。
  5. プラグインを追加 > カスタムプラグインをクリックします。
  6. [ プラグイン名 ] に、と入力しますsimulivePlugin。
  7. JavaScript URLおよび CSS URL については、アカウントマネージャーから提供された値を使用します。
  8. [保存] をクリックして、変更を保存します。
注:プレイヤーに加えた変更は、公開されているプレイヤーに表示されるまでに最大 5 分かかる場合があります。

カスタムプレーヤーの設定

カスタムプレーヤーを使用するように仮想イベントエクスペリエンスを構成するには、次の手順に従います。

  1. ギャラリーモジュールを開きます。
  2. エクスペリエンスを編集します。
  3. クリックビデオと再生>プレーヤーとリードフォーム左側のナビゲーションで。
  4. [ サイト固有のプレーヤーを使用]オプションを選択し、Simulive プラグインで設定されたプレーヤーを選択します。
  5. [ 保存] をクリックします。

Simulive カスタムフィールドを作成する

カスタムフィールドは、ビデオが使用可能になる日付と時刻を格納するために使用されます。Simulive カスタムフィールドを作成するには、次の手順を実行します。

  1. ナビゲーションヘッダーで、[ 管理 ]、[ ビデオフィールド] の順にクリックします。管理者メニューにアクセスするには、アカウント管理者である必要があります。
  2. [ カスタムフィールドの追加] をクリックします。
  3. 次の値を使用して、新しいカスタムフィールドを作成します。
    • 表示名 -Simulive スタート
    • 内部名 -simulive_start
    • タイプ -テキスト
    • 説明 -予想されるフォーマットは YYYY-MM-DDTHH: MM: SS-HH: MM です
  4. [ フィールドを保存] をクリックします。
カスタムフィールドの作成については、「カスタムメタデータフィールドの作成」を参照してください。

Simuliveビデオプロパティの設定

カスタムフィールド Simulive Startには、ビデオが利用可能になる日時が格納されます。Simulive 開始時刻に達するまで、ビデオは閲覧できません。シミュレーション動画にもサイマルタグを付ける必要があります。ビデオを Simulive ビデオとして設定するには、次の手順に従います。

  1. Media モジュールを開きます。
  2. ビデオタイトルをクリックしてプロパティを開きます。
  3. [ ビデオプロパティ]セクションを見つけて、[ 編集] をクリックします。
  4. ビデオにsimuliveタグを付けます。
  5. [保存]をクリックします。
  6. [ カスタムフィールド]セクションを見つけて、[ 編集] をクリックします。
  7. [ シミュレーション開始]フィールドに日付と時刻の値を入力します。フィールドの正しい形式は、必要に応じて先頭にゼロが付いた年、月、YYYY-MM-DDTHH:MM:SS-HH:MMYYYY-MM-DD日付をどこにあるかです。HH:MM:SSは、24 時間の時間、分、秒を適宜先頭にゼロで指定します。また、-HH:MMタイムゾーンのオフセットです（UTC-4 の場合は-04:00、UTC+8.5 の場合は + 08:30 など）。
    注:Simulive は、ローカルコンピューター時刻と構成された Simulive Start Time を比較して、特定のユーザーのローカルコンピューター時間に基づいてイベントをいつ開始するかを決定します。このため、一部のユーザーでは、イベントが予想よりも早く開始されることがあります。これは、通常、' これらのユーザーのコンピュータの時間が不正確であるためです。
  8. [保存]をクリックします。

ポリシーキー値の更新

カウントダウンタイマーをシミュレーションビデオに表示するには、設定ファイルのポリシーキー値を検索が有効なポリシーキーで更新する必要があります。これを行わないと、動画にカウントダウンが表示されなくなります。検索が有効なポリシーキーを取得する手順については、「コードサンプル:ポリシーキーを生成します。新しいポリシーキーが作成されたら、policyKey設定ファイルの値を更新し、エクスペリエンスを再公開します。
