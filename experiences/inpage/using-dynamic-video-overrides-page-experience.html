ページはめ込みエクスペリエンスでの動的ビデオオーバーライドの使用

{{ page.description }}

ページ内エクスペリエンスで動画を変更したい場合があります。たとえば、エクスペリエンスの動画は、ユーザーが選択した内容に基づいて、またはサイトの他の部分に移動した内容に基づいて変更する必要がある場合があります。それぞれの動画セットが異なるページ内エクスペリエンスを作成する代わりに、埋め込みコードを変更してエクスペリエンスに適切な動画を割り当てることができます。埋め込みコードの変更は、使用されている埋め込みコードの種類によって異なります。

  • 推奨埋め込みコード
  • JavaScript の埋め込みコードはありません

推奨埋め込みコード

一般的なページ内エクスペリエンスの埋め込みコード（推奨オプション）は、次のようになります。

<div data-experience="597a2aa114b2ed00103052af"></div>
<script src="https://players.brightcove.net/20318290001/experience_597a2aa114b2ed00103052af/live.js"></script>

ページ内エクスペリエンスに関連付けられた動画を動的に上書きするには、埋め込みコードを変更して、div使用する動画またはプレイリストを指定する属性をタグに追加します。

個々の動画の指定

使用する動画を個別に指定するには、data-video-idsdivパラメーターをタグに追加し、ビデオ ID のカンマ区切りリストを指定します。

<div data-experience="597a2aa114b2ed00103052af" 
     data-video-ids="12345,56789"></div>
<script src="https://players.brightcove.net/20318290001/experience_597a2aa114b2ed00103052af/live.js"></script>

プレイリストの指定

使用するプレイリストを指定するには、data-playlist-iddivパラメーターをタグに追加し、使用するプレイリスト ID を指定します。

<div data-experience="597a2aa114b2ed00103052af" 
     data-playlist-id="12345"></div>
<script src="https://players.brightcove.net/20318290001/experience_597a2aa114b2ed00103052af/live.js"></script>

JavaScript 埋め込みコードはありません

一般的なページ内エクスペリエンスの埋め込みコード（JavaScript オプションなし）は、次のようになります。

<iframe style="display:block;border:none;margin-left:auto;margin-right:auto;"
src="https://players.brightcove.net/20318290001/experience_597a2aa114b2ed00103052af/index.html"
allowfullscreen webkitallowfullscreen mozallowfullscreen></iframe>

ページ内エクスペリエンスに関連付けられた動画を動的に上書きするには、src埋め込みコードを変更して属性にパラメータを追加できます。

個々の動画の指定

使用する動画を個別に指定するには、videoIdsパラメーターを追加し、ビデオ ID のカンマ区切りリストを指定します。

<iframe style="display:block;border:none;margin-left:auto;margin-right:auto;"
src="https://players.brightcove.net/20318290001/experience_597a2aa114b2ed00103052af/index.html?videoIds=12345,56789""
allowfullscreen webkitallowfullscreen mozallowfullscreen></iframe>

プレイリストの指定

使用するプレイリストを指定するには、playlistIdパラメーターを追加し、使用するプレイリスト ID を指定します。

<iframe style="display:block;border:none;margin-left:auto;margin-right:auto;"
src="https://players.brightcove.net/20318290001/experience_597a2aa114b2ed00103052af/index.html?playlistId=12345"
allowfullscreen webkitallowfullscreen mozallowfullscreen></iframe>
