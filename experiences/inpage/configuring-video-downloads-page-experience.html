ページはめ込みエクスペリエンス用のビデオダウンロードの構成

{{ page.description }}

ギャラリーインページエクスペリエンスは、視聴者が動画コンテンツをダウンロードできるようにダウンロードリンクを表示するように設定できます。ページ内エクスペリエンスは、カスタムフィールド値に基づいて、すべての動画または特定の動画のダウンロードリンクが表示されるように設定できます。動画をダウンロードすると、動画の横にダウンロードリンクが表示されます。利用可能な最高品質のレンディションがダウンロードされます。

注:iOS デバイスでは、ビデオのダウンロードはサポートされていません。

エクスペリエンス内のすべてのビデオのダウンロードリンクを提供するには、「ページ内エクスペリエンスの設定の構成」を参照してください。指定した動画のみを選択的にダウンロードできるようにするには、次の手順を実行する必要があります。

  1. ダウンロード可能なコンテンツを示すカスタムフィールドを作成する
  2. カスタムフィールドに適切な値を割り当てる
  3. ダウンロードを許可するようにページ内エクスペリエンスを構成する

ダウンロードコンテンツを示すカスタムフィールドの作成

ダウンロードできるビデオは、カスタムフィールドの値を設定することによって示されます。まず、動画をダウンロードできることを示すために使用するカスタムフィールドを、アカウントに作成します。

  1. ビデオクラウドスタジオにログインします。
  2. ナビゲーションヘッダーで、[ 管理 ]、[ ビデオフィールド] の順にクリックします。管理者メニューにアクセスするには、アカウント管理者である必要があります。
  3. [ カスタムフィールドの追加] をクリックします。
  4. ビデオをダウンロードできるかどうかを示すために使用する新しいカスタムフィールドを作成します。割り当てられた名前と値は重要ではありません。次の例では、ダウンロード可能な新しいカスタムフィールドが作成されています。[ タイプ]が [ リスト] に設定され、2 つの値 ( trueと false)が設定されます。ダウンロード可能な動画には動画プロパティの一部として、この値が true に設定されます。
  5. [ 変更を保存] をクリックし、[ 保存] をクリックします。

カスタムフィールド値の割り当て

ダウンロードできるビデオには、作成したカスタムフィールドに値が割り当てられている必要があります。

  1. メディアモジュールを開きます。
  2. ビデオをダウンロード可能としてマークするには、ビデオ名をクリックしてビデオのプロパティを開きます。
  3. [ カスタムフィールド]セクションを見つけて、[ 編集] をクリックします。
  4. 必要に応じて、カスタムフィールドの値を設定します。この例では、ダウンロード可能に設定されていますNS。
  5. [保存]をクリックします。
  6. 必要に応じて、他の動画の動画プロパティを編集します。
注:メディアモジュールのクイック編集機能を使用して、複数のビデオのカスタムフィールド値を同時に編集します。

ダウンロードを許可するためのページ内エクスペリエンスの設定

カスタムフィールド値に基づいてビデオのダウンロードを許可するようにページ内エクスペリエンスを設定するには、次の手順に従います。

  1. ギャラリーモジュールを開きます。
  2. 自分のエクスペリエンスを見つけて、[ 編集] をクリックします。
  3. クリック設定>ビデオ左側のナビゲーションで。
  4. [ ビデオのダウンロード]セクションで、[ カスタムフィールド値に基づいて各動画のダウンロードリンクを提供する]オプションを選択します。
  5. 上記の手順 4 で作成したカスタムフィールド名と値を入力します。この例では、作成したカスタムフィールドの名前は downloadable であり、ダウンロード可能な動画の値を true に設定しました。
  6. [ 保存 ] をクリックし、エクスペリエンスを再公開します。
  7. ダウンロード可能なカスタムフィールドが true に設定されている動画のダウンロードリンクが表示されることを確認します。
