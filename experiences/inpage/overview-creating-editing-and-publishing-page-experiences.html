概要:ページ内エクスペリエンスの作成、編集、公開

{{ page.description }}

ページ内エクスペリエンスは、自分のサイトのページに含まれており、オーディエンスを惹きつける基本的なインタラクティブ機能（リンクやカード）を含めることができます。

注:ギャラリーでは、指定された URL でアクセスされるスタンドアロンの Web サイトであるポータルエクスペリエンスを作成することもできます。詳細については、「概要」を参照してください。ポータル・エクスペリエンスの作成、編集、公開。

以下は、ギャラリーを使用して構築されたページ内エクスペリエンスの例です。ページ内エクスペリエンスは Web ページに埋め込まれ、他のページコンテンツとともに表示されます。

Brightcove Gallery を使用してページ内エクスペリエンスを構築するためのステップバイステップガイドについては、「ステップバイステップ:ページ内エクスペリエンスの作成と公開。

ページ内エクスペリエンスを作成および公開する主な手順は次のとおりです。

  • テンプレートを選択
  • 動画を追加してエクスペリエンスをカスタマイズする
  • エクスペリエンスをプレビューして公開する
  • エクスペリエンス分析のレビュー

テンプレートの選択

新しいギャラリーエクスペリエンスを作成するときは、まずテンプレートを選択します。テンプレートは、エクスペリエンスの全体的なルックアンドフィールを制御します。Gallery には、さまざまな種類のエクスペリエンスを提供できる柔軟性を提供する一連のテンプレートが用意されています。

  • ポータル
  • ページ内
  • ランディングページ
  • イベント

ページ内エクスペリエンステンプレートの詳細およびサンプルエクスペリエンスの表示については、「ギャラリーインページエクスペリエンステンプレートの概要」トピックを参照してください。

Gallery には、作成するエクスペリエンスの種類に基づいてテンプレートをフィルタリングする機能があります。ページ内エクスペリエンステンプレートを選択したら、エクスペリエンスのタイトル、説明、サイズを設定できます。

ビデオの追加とエクスペリエンスのカスタマイズ

エクスペリエンスが作成されると、レイアウトエディターは次の機能を提供します。

  • テキストと画像でエクスペリエンスをカスタマイズする
  • エクスペリエンスに動画を追加する
  • 使用するフォントと色を選択して、エクスペリエンスのスタイルを設定します。
  • エクスペリエンスの設定をカスタマイズする
  • エクスペリエンスの詳細とサイズを編集する

エクスペリエンスのプレビューと公開

ページ内エクスペリエンスをプレビューして、デスクトップ、タブレット、モバイルデバイスでの表示方法を確認できます。エクスペリエンスを公開すると、プレビュー URL と Web サイトにエクスペリエンスを埋め込むために必要な発行コードが生成されます。詳細については、「ページ内エクスペリエンスのプレビューと公開」を参照してください。

エクスペリエンス分析の確認

ページ内エクスペリエンスを公開すると、Gallery はこれらのエクスペリエンスの分析を収集し、さまざまなレポートで利用できるようになります。詳細については、「ページ内エクスペリエンス分析のレビュー」を参照してください。
