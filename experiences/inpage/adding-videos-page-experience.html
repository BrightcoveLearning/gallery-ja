ページ内エクスペリエンスへのビデオの追加

{{ page.description }}

Video Cloud 動画と YouTube 動画は、ページ内エクスペリエンスに追加できます。ページはめ込みエクスペリエンスに動画を追加するには、エクスペリエンスを編集してから[ ビデオ左側のナビゲーションで。

注:YouTube 動画をエクスペリエンスに追加するには、YouTube の設定を構成する必要があります。

[ ビデオを追加] をクリックしてビデオセレクターを開き、[ ソース] を選択します。

  • Video Cloud -ビデオクラウドの動画コンテンツが使用されます
  • YouTube -YouTube 動画コンテンツが使用されます（詳細については、このドキュメントの YouTube統合の詳細セクションを参照してください）
注:Video Cloud と YouTube の両方の動画を同じページ内エクスペリエンスに追加できます。

YouTubeソースが選択されている場合は、チャンネルセレクターが表示され、使用する YouTube チャンネルを選択できます。

[Video Cloud Source ] を選択したら、動画を追加する方法を選択します。

  • 動画を選択 -ビデオは手動で選択して使用されます
  • プレイリストを選択 -プレイリストの動画が使用されます
  • 参照 ID を使用 -プレイリストの参照 ID のビデオが使用されます

ビデオが選択されると、ページの本文に表示されます。この例では、Video Cloud プレイリストが使用されています。

注:エクスペリエンスに変更を加えたら、エクスペリエンスを再公開して、エクスペリエンスを表示するサイトに変更が表示されるようにしてください。

動画を手動で選択する

ビデオを手動で選択するには、[ ビデオの選択方法] を選択します。Video Cloud ソースを使用すると、Video Cloud アカウント内で最近更新された 100 本の動画のリストが動画セレクタに表示されます。ページダウンすると、新しい動画が取得されます。YouTube ソースでは、選択したチャンネル内のすべての動画が表示されます。ビデオをクリックして選択し、もう一度クリックして選択を解除します。検索フィールドを使用して動画を検索します。この例では、Video Cloud ソースが選択されています。

完了したら、[ 追加] をクリックします。

動画の検索

[ 検索 ] フィールドは、動画を検索するために使用できます。Video Cloud 動画を検索する場合、ギャラリーは動画名、短い説明と長い説明、タグ、カスタムフィールドを検索します。YouTube 動画を検索する場合、ギャラリーは YouTube 検索 API を使用します。

[ その他の検索オプション ] をクリックすると、検索するフィールドをより詳細に制御したり、テキストを含めたり除外したりできます。このオプションは、Video Cloud ソースを使用している場合にのみ使用できます。

プレイリストの使用

再生リストで動画を使用するには、方法のプレイリストを選択。Video Cloud ソースを使用すると、Video Cloud アカウントで最近更新された 20 個の再生リストのリストが動画セレクタに表示されます。YouTubeソースでは、ビデオセレクタには、選択したチャンネル内のすべての再生リストのリストが表示されます。[検索] フィールドを使用してプレイリストを検索できます。この例では、Video Cloud ソースが選択されています。

選択できるプレイリストは 1 つのみであることに注意してください。完了したら、[ 選択] をクリックします。

プレイリストの参照 ID を使用する

プレイリストの参照 ID を指定してプレイリスト内のビデオを使用するには、[参照 ID の使用方法] を選択します。このオプションは、Video Cloud ソースを使用している場合にのみサポートされます。ビデオセレクターには、プレイリストのリファレンス IDを入力するためのフィールドがあります。プレイリストの参照 ID を使用すると、Media モジュールを使用して参照 ID を別のプレイリストに再割り当てすることで、エクスペリエンスの動画を簡単に変更できます。 エクスペリエンスに表示されるビデオを変更する必要がある場合は、エクスペリエンスを編集して再公開する必要はありません。

使用できるプレイリスト参照 ID は 1 つだけです。完了したら、[ 選択] をクリックします。

ビデオへのインタラクティブ性の追加

インタラクションコンポーネントとコンパニオンコンポーネントをビデオに追加し、ビデオの再生時に表示できます。インタラクションにはリンクやカードが含まれており、動画の再生時にプレイヤーの一部として表示されます。コンパニオンコンポーネントには HTML、画像、テキストオブジェクトが含まれており、プレーヤーの外に表示されます。

インタラクティブ機能を使用して構成されたビデオは、双方向性相互作用のタイプを示す列。次の例では、リンクとカードのインタラクションとコンパニオンイメージが設定されています。

インタラクションおよびコンパニオンコンポーネントの追加と構成については、次を参照してください。

  • ページ内エクスペリエンスにインタラクションを追加する
  • ページ内エクスペリエンスのコンパニオンコンポーネントの構成

ページ内エクスペリエンスから動画を削除する

インページエクスペリエンスから動画を削除するプロセスは、動画を追加するために選択した方法によって異なります。

  • ビデオ -削除するビデオの横にあるチェックボックスをオンにして、[ ビデオの削除] をクリックします。削除アイコンをクリックすることもできます（) の [ アクション ] 列に表示されます。
  • プレイリストを選択 -[ プレイリストの削除] をクリックして、すべての動画を削除します。
  • 参照 ID を使用 -[ プレイリストの削除] をクリックして、すべての動画を削除します。

動画を削除しても、そのビデオに設定されたインタラクションやコンパニオンコンポーネントは削除されません。その後、ビデオがエクスペリエンスに再び追加されると、インタラクションとコンパニオンコンポーネントが再表示されます。

YouTube 統合の詳細

  • YouTubeの動画をダウンロードできない
  • YouTube 動画を動画サイトに追加する前に、YouTube 設定を構成する必要があります
  • YouTube 動画のプライバシー設定を [公開] に設定する必要があります
  • 現在、ギャラリーでは YouTube 動画の分析データは収集されません。ページ内エクスペリエンスの YouTube 動画では、エクスペリエンス分析ダッシュボードにアナリティクスデータが表示されない
  • IMAプラグインを備えたBrightcoveプレーヤー内部で再生されるYouTube動画は、モバイルデバイスで広告を表示しません
  • お客様の YouTube' の使用には YouTube の利用規約が適用されます。
  • 検索する場合、すべてのコレクションは YouTube 動画で構成され、動画は選択したチャンネルによって所有されている必要があります
