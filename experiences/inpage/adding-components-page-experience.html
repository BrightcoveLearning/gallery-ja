    ページ内エクスペリエンスへのコンポーネントの追加

    このトピックでは、ページ内エクスペリエンスにコンポーネントを追加する方法について説明します。

    コンポーネントをページ内エクスペリエンスに追加して、エクスペリエンスをカスタマイズし、ブランド化することができます。ページ内エクスペリエンスにコンポーネントを追加するには、エクスペリエンスを編集し、左側のナビゲーションで [ レイアウト ] をクリックします。

    コンポーネントを追加するプラスアイコン ( ) をクリックし、追加するコンポーネントのタイプを選択します。

    次のタイプのコンポーネントを追加できます。

    • 広告
    • チャット
    • カスタム HTML
    • イメージ
    • テキスト
    • 3Play
    • Twitterフィード
    注：エクスペリエンスに何らかの変更を加えたら、エクスペリエンスが表示されるサイトに変更が表示されるように、エクスペリエンスを再公開してください。

    アドバタイズメントコンポーネントの追加

    エクスペリエンスに広告を追加するには:

    1. コンポーネントピッカーで [ アドバタイズ ] をクリックします。
    2. 提供情報の [ 名前 ] を入力します。
    3. ヘッダー広告コードと広告ユニットコードを入力します。ヘッダー広告コードは、エクスペリエンス内のすべての広告で共有されます。
    4. 広告を表示するタイミングを選択します。
    5. [保存]をクリックします。広告はエクスペリエンスに表示されます。
    注：複数の広告でページはめ込みエクスペリエンスを使用している場合は、各広告ユニットコードに一意のdivIDがあることを確認してください。必要に応じて、の ID divとコンテナパラメータを編集して、その ID adblock1と一致させる必要があります。

    チャットコンポーネントの追加

    サードパーティのチャットウィジェットをエクスペリエンスに追加して、チャット機能を提供できます。エクスペリエンスにチャットコンポーネントを追加するには:

    1. コンポーネントピッカーで [ チャット ] をクリックします。
    2. チャットコンポーネントの名前を入力します。
    3. チャットの HTML コードを入力します。
    4. チャットを表示するタイミングを選択します。
    5. [保存]をクリックします。チャットはエクスペリエンスにレンダリングされます。

    カスタム HTML コンポーネントの追加

    カスタムコンポーネントを使用して、独自のカスタム HTML をエクスペリエンスに追加できます。たとえば、マーケティングオートメーションプラットフォーム (MAP) を使用してカスタムリードフォームを作成した場合、カスタム HTML コンポーネントを使用して MAP のリードフォーム HTML をインページエクスペリエンスに追加できます。

    エクスペリエンスにカスタム HTML を追加するには:

    1. コンポーネントピッカーで [ カスタム ] をクリックします。
    2. HTML コンポーネントの「名前」を入力します。
    3. エクスペリエンスに追加するカスタム HTML を入力します。
    4. HTML を表示するタイミングを選択します。
    5. [保存]をクリックします。HTML はエクスペリエンスにレンダリングされます。

    HTML が入力されていない場合は、HTML プレースホルダが表示されます。

    注：ギャラリーオーサリング環境では、セキュリティ上の理由から、カスタム HTML コンポーネントからスクリプトを削除します。このスクリプトは、ページ内エクスペリエンスがプレビューおよび公開されたときに実行されます。これにより、カスタム HTML コンポーネントの外観が異なる可能性があります。特に、JavaScript コンテンツが多い場合は特にそうです。

    イメージコンポーネントの追加

    エクスペリエンスにイメージを追加するには:

    1. コンポーネントピッカーで [ イメージ ] をクリックします。
    2. イメージの名前を入力します。
    3. イメージを指定するには、次の操作を実行します。
      • ファイルシステムからイメージをドラッグし、ドロップ領域にドロップします
      • [ 参照 ] をクリックし、ファイルシステムからイメージを選択します。
      • リモートイメージファイルの URL を入力してください
    4. イメージのサイズを設定します。
      • 画像はコンテナにサイズ変更される -コンテナより大きいサイズの画像は、収まるように縮小されます。小さい画像は拡大されません。
      • 幅と高さ : イメージの幅と高さを手動で入力します。
    5. イメージの位置合わせを設定します。
    6. クリックスルー URL を入力します。これは、画像がクリックされたときに移動する URL です。リンクを新しいウィンドウで開くこともできます。
    7. 画像を表示するタイミングを選択します。
    8. [ 保存 ] をクリックして、イメージをエクスペリエンスに追加します。

    画像が選択されていない場合は、画像プレースホルダが表示されます。

    文字コンポーネントの追加

    エクスペリエンスにテキストを追加するには:

    1. コンポーネントピッカーで [ テキスト ] をクリックします。
    2. エクスペリエンスに追加するテキストを入力します。ツールバーを使用して、テキストの書式を設定できます。
    3. テキストの名前を入力します。
    4. テキストを表示するタイミングを選択します。
    5. [ 保存 ] をクリックしてテキストを追加します。テキストがエクスペリエンスに表示されます。

    テキストを入力しない場合は、テキストプレースホルダが表示されます。

    3Play コンポーネントの追加

    3Play コンポーネントを使用して、3Play インタラクティブトランスクリプトをエクスペリエンスに追加できます。このコンポーネントは、ビデオキャプションが 3Play Media によって生成された場合にのみ機能します。このコンポーネントがエクスペリエンスに追加されると、 [インタラクティブトランスクリプトを表示] リンクが表示されます。クリックすると、ビデオトランスクリプトが表示されます。次の例では、3Play コンポーネントがビデオプレーヤーの下に追加され、再生中の状態で表示されるように構成されています。

    実例については、このリンクを参照してビデオを再生してください。

    注：アカウントで 3Play コンポーネントを有効にするには、アカウントマネージャーまたは Brightcove サポートにお問い合わせください。

    エクスペリエンスに 3Play コンポーネントを追加するには:

    1. コンポーネントピッカーで [ 3再生 ] をクリックします。
    2. 3Play コンポーネントの名前を入力します。
    3. 3Play プロジェクト ID を入力します。3Play プロジェクト ID を確認するには、3Play アカウントにログインし、ページヘッダーの [ アカウントの概要 ] リンクをクリックします。そのページにプロジェクト ID が表示されます。
    4. 3Play コンポーネントを表示するタイミングを選択します。
    5. 「保存」をクリックして、3Play コンポーネントを追加します。3Play コンポーネントがエクスペリエンスに表示されます。

    Twitter フィードコンポーネントの追加

    エクスペリエンスにTwitterフィードを追加するには:

    1. コンポーネントピッカーで [ Twitter フィード ] をクリックします。
    2. Twitterフィードの名前を入力します。
    3. Twitterサイトでウィジェットを作成します。Twitterウィジェットの作成に関するヘルプは「Twitter開発者ドキュメント」を参照してください。
    4. TwitterウィジェットのHTMLをコピーし、Twitter Feed設定に貼り付けます。
    5. Twitterフィードを表示するタイミングを選択します。
    6. [ 保存 ] をクリックして Twitter フィードを追加します。Twitterフィードがエクスペリエンスに表示されます。
