    ページ内エクスペリエンスへのインタラクションの追加

    このトピックでは、ページ内エクスペリエンスの動画にカードとリンクインタラクションを追加する方法について説明します。

    ビデオは、カードとリンクのインタラクションで構成できます。ビデオの再生中に、インタラクションがプレーヤーの上に表示されます。次の例では、プレーヤーの左下にリンクが表示され、右上にカードが表示されています。カードには、ユーザーがクリックしてカードを開くことができる情報アイコン ( ) が表示されます。

    一部のページ内エクスペリエンスは、コンパニオンコンポーネントもサポートしています。コンパニオンコンポーネントを使用すると、再生中のビデオに基づいてプレーヤー外のコンテンツを変更できます。たとえば、商品ビデオを再生すると、プレーヤーの右側にクリック可能な画像があり、その商品の詳細が表示されます。コンパニオンコンポーネントはインタラクションに似ています。違いは、インタラクションがビデオプレーヤー自体上に表示されます。コンパニオンコンポーネントの詳細は、「ページ内エクスペリエンスのコンパニオンコンポーネントの構成」を参照してください。

    動画にインタラクションを追加するには、エクスペリエンスを編集し、左側のナビゲーションで [ 動画 ] をクリックします。ビデオにインタラクションを追加するには、ビデオを選択し、[ インタラクティブ性を追加 ] をクリックしてから、追加するインタラクティビティの種類をクリックするか、リンク ( ) またはカード ( ) インタラクティブアイコンをクリックしますビデオに関連付けられています。

    このトピックでは次の項目について説明します。

    • リンクインタラクションを追加する
    • カードインタラクションを追加する
    • 対話の編集
    • インタラクションを削除する
    注：コンパニオン HTML、画像、テキストインタラクションの追加については、ページ内エクスペリエンスのコンパニオンコンポーネントの構成を参照してください。
    注：エクスペリエンスに何らかの変更を加えたら、エクスペリエンスが表示されるサイトに変更が表示されるように、エクスペリエンスを再公開してください。

    リンクインタラクションを追加する

    動画にリンクインタラクションを追加するには:

    1. ビデオをクリックして選択します。
    2. [ > インタラクティブリンクを追加 ] をクリックするか、[ アクション ] 列のリンク ( ) アイコンをクリックします。インタラクティブエディターが開きます。
    3. リンクテキストとリンク URL を追加します。値が入力されると、プレーヤーのリンクプレビューが更新されます。プレーヤーの下のスライダーは、リンクの開始時間と期間を示します。
    4. 再生動作を選択します。
      • なし -リンクがクリックされたときにビデオが再生され続けます
      • クリック時に一時停止 -リンクがクリックされると動画が一時停止します
    5. リンク開始時刻と期間を構成するには、[ Configure ] リンクをクリックします。
    6. 開始時刻と期間を選択します。
      • 動画全体を表示-動画全体のリンクが表示されます
      • 開始時刻と期間を選択 -リンクの開始時刻と期間を手動で設定します。
    7. リンクの開始時間と期間を設定します。
      注：リンクの長さは、プレーヤーの下のスライダーを使用して設定することもできます。左矢印をリンクが表示されるタイミングまでドラッグし、右矢印をリンクが消えるタイミングまでドラッグします。
    8. 開始時刻のみを設定し、動画が終了するまでリンクを表示する場合は、[動画が終了するまで] をオンにします。
    9. エクスペリエンス内のすべての動画にこのリンクのインタラクションを表示する場合は、[レイアウト内のすべての動画にこのインタラクションを適用する ] をオンにします。
    10. リンクの配置を設定するには、[ 設計 ] リンクをクリックします。
    11. プレースメントをクリックして、リンクがプレイヤーに表示される場所を指定します。
    12. [保存] をクリックして、変更を保存します。
    注：インタラクションを別の動画に割り当てるには、プレーヤーの上にあるビデオセレクタドロップダウンリストを使用して、別のビデオを選択します。

    カードインタラクションを追加する

    カードインタラクションをビデオに追加するには:

    1. ビデオをクリックして選択します。
    2. [ > インタラクティブカードを追加 ] をクリックするか、[ アクション ] 列のカードアイコン ( ) をクリックします。インタラクティブエディターが開き、カードのプレビューがプレーヤーに表示されます。
    3. カードに画像を追加するには、次の操作を実行します。
      • 画像をドラッグし、ドロップエリアにドロップします。
      • [ 参照 ] をクリックし、ファイルシステムからイメージを選択します。
      • テキストボックスにリモートイメージフィールドへの URL を入力します
    4. カードタイトル、説明、リンクテキスト、リンクURLを追加します。値が入力されると、カードのプレビューが更新されます。プレーヤーの下のスライダーは、カードの開始時間と期間を示します。
    5. 再生動作を選択します。
      • なし -カード内のリンクをクリックしてもビデオは再生され続けます
      • 開いたときに一時停止 -カードが開いたときにビデオが一時停止します
      • リンククリック時に一時停止 -カード内のリンクがクリックされると動画が一時停止します
    6. カードを自動的に表示するには、[カードを自動的に開く ] を選択します。
    7. カードの開始時間と時間を設定するには、[ Configure ] リンクをクリックします。
    8. 開始時刻と期間を選択します。
      • ビデオ全体を表示 -カードは、ビデオ全体に表示されます
      • 開始時間と期間を選択 -カードの開始時間と期間を手動で設定します。
    9. カードの開始時間と期間を設定します。
      注：カードの持続時間は、プレーヤーの下のスライダーを使用して設定することもできます。左矢印をカードが表示されるタイミングまでドラッグし、右矢印をカードが消えるまでドラッグします。
    10. 開始時間のみを設定し、ビデオが終了するまでカードを表示する場合は、[ビデオが終了するまで] をオンにします。
    11. エクスペリエンスのすべての動画にこのカードのインタラクションを表示する場合は、[このインタラクションをレイアウト内のすべての動画に適用する ] をオンにします。
    12. [保存] をクリックして、変更を保存します。
    注：インタラクションを別の動画に割り当てるには、プレーヤーの上にあるビデオセレクタドロップダウンリストを使用して、別のビデオを選択します。

    対話の編集

    インタラクションが設定されたビデオは、インタラクションの種類を示すアイコンが [ インタラクティビティ ] 列に表示されます。

    注：コンパニオンアイコン（）は、ビデオにコンパニオンコンポーネントがあることを示すこともできます。コンパニオンコンポーネントの詳細は、「ページ内エクスペリエンスのコンパニオンコンポーネントの構成」を参照してください。

    [ インタラクティビティ ] 列のアイコンの上にカーソルを置くと、そのタイプのすべてのインタラクションが表示されます。この例では、2 つのリンクが設定されています。

    ビデオのインタラクションを編集するには、次の手順に従います。

    1. [ インタラクティビティ ] 列のアイコンにカーソルを合わせ、インタラクションの横にある [編集] ( ) アイコンをクリックするか、[ アクション ] 列の編集アイコン ( ) をクリックするか、ビデオタイトルをクリックします。インタラクティブエディターが開きます。
    2. 必要に応じて、インタラクションを編集します。
    3. [保存] をクリックして、変更を保存します。

    インタラクションを削除する

    インタラクションを削除するには、[ インタラクティビティ ] 列のアイコンにカーソルを合わせ、[削除] ( ) アイコンをクリックします。

    注：[ アクション ] 列の削除アイコンをクリックすると、エクスペリエンスから動画が削除されます。

    [ アクション ] 列の編集アイコン ( ) をクリックしてすべてのインタラクションのリストを表示し、適切な削除 ( ) アイコンをクリックすることもできます。

    注：ページ内エクスペリエンスから動画を削除しても、その動画に設定されているインタラクションは削除されません。その後、ビデオがエクスペリエンスに再度追加されると、インタラクションが再び表示されます。
