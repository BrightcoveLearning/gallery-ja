ページ内エクスペリエンスのCall-to-Action分析の確認

{{ page.description }}

行動を促す分析は、インページエクスペリエンスのクリックスルーとフォームコンバージョンに関する分析を提供します。行動を促す分析を確認するには、エクスペリエンスを編集し、左側のナビゲーションで [ エクスペリエンス分析] > [アクションへの呼び出し ] の順にクリックします。

タイムスパンの選択

レポートの期間を指定するには、クイック日付をクリックするか、カレンダーコントロールを使用します。

クイック日付を使用する

一連のクイック日付がページの上部に表示されます。次のクイック日付を使用できます。

  • 今日
  • 昨日
  • 過去 7 日間
  • 過去 30 日間
  • 今月 -現在までの暦月-たとえば、今日が 3 月 17 日である場合、このレポートには 3 月 1 ～ 3 月 17 日が表示されます。
  • 先月 -前の暦月-たとえば、今日が 2 月 14 日である場合、このレポートは 1 月 1 日から 1 月 31 日と表示されます。

レポートを更新するには、クイック日付をクリックします。

カレンダーコントロールの使用

カレンダーコントロールをクリックして、開始日と終了日を選択することもできます。

[ 適用 ] をクリックして、レポートを更新します。

クリックスルー分析の確認

クリックスルー分析は、ページ内エクスペリエンスのインタラクティブ性を追跡します。次のオブジェクトは、クリックスルー分析をトリガーして追跡します。

  • リンク URL で設定されたリンクインタラクション
  • リンク URL で構成されたカード操作
  • クリック可能な URL で構成された画像コンポーネント

クリックスルー分析を確認するには、をクリックします経験分析>召喚状左側のナビゲーションでクリックしますクリックスルーページヘッダー内。

クリックスルー分析では、エクスペリエンスの次のデータが表示されます。

  • 行動を促す名前 -アクションを促すコールの名前:
    • リンク-リンクタイトルを表示します
    • カード-カードタイトルを表示します
    • Images-作成時に割り当てられた名前を表示します。
    • カスタム HTML-作成時に割り当てられた名前を表示します。
  • 更新 -インタラクションの最終更新日時
  • タイプ -インタラクションの種類、リンクまたはカード
  • 印象 -インタラクティビティが表示された回数（カードの場合、カードアイコンが表示されたときにインプレッションが記録されます（）が表示されます。インプレッションを記録するためにカードを表示する必要はありません）
  • Clickthrough -リンクがクリックされた回数、カード内のリンクがクリックされた回数、または画像コンポーネントのクリックスルー URL がクリックされた回数
  • クリック率（%） -リンクをクリックした視聴者の割合（クリックスルー数をインプレッション数で割った割合）

フォームコンバージョン分析の確認

フォームコンバージョン分析は、インページエクスペリエンスのフォームの送信とカスタム HTML リンクアクティビティを追跡します。次のオブジェクトは、フォームコンバージョン分析のトラッキングをトリガーします。

  • HTML コンパニオンコンポーネントから送信されたフォーム
  • カスタム HTML コンポーネントから送信されたフォーム
  • カスタム HTML コンポーネント内のクリック可能な URL

フォーム変換分析を確認するには、をクリックします経験分析>召喚状左側のナビゲーションでクリックしますフォーム変換ページヘッダー内。

フォームコンバージョン分析には、エクスペリエンスの次のデータが表示されます。

  • Call-to-Action Name -コンポーネントの名前（コンポーネントがエクスペリエンスに追加されたときに名前が割り当てられます）
  • Updated -コンポーネントの最終更新日時
  • タイプ -インタラクションのタイプ（HTMLコンパニオンコンポーネントは（）アイコン
  • インプレッション -コンポーネントが表示された回数
  • フォーム変換 -フォームが送信された回数、またはカスタム HTML コンポーネント内の URL がクリックされた回数
  • フォームのコンバージョン率（%） -フォームを送信した視聴者の割合（フォームコンバージョンをインプレッション数で割った割合）
