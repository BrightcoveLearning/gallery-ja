Brightcove Gallery ー製品アップデート

リリースノートは、次のものでも入手できます。

  • ギャラリーテンプレートリリースノート

SSO 認証コンテキスト要求のディセーブル化

2021年4月15日

SSO 認証コンテキスト要求を無効にする新しいインターフェイスオプションがアクセス制御プロファイルに追加されました。詳細は、「アクセス制御プロファイルの設定」を参照してください。

ビデオの自動再生設定

  2019年9月4日

新しい動画の自動再生設定が追加され、詳細ページでの動画の再生動作を制御できます。詳細については、「ポータルエクスペリエンスのプレーヤーおよびリードフォーム設定のカスタマイズ」を参照してください。

ギャラリーナビゲーションの再設計

  2018年9月27日

ポータルエクスペリエンスの編集時のナビゲーションメニューが再設計され、ページ内エクスペリエンスの編集環境との整合性が向上しました。詳細については、「サイトエディタを使用してポータルエクスペリエンスをカスタマイズする」を参照してください。

ギャラリーホームページに検索オプションを追加しました

2018年8月1日

ギャラリーのホームページに「検索」ボックスが追加され、エクスペリエンスのリストを検索できるようになります。検索時には、次のプロパティが検索されます。

  • タイトル
  • 説明
  • ID
  • テンプレート
  • URL

ページ内エクスペリエンスへのビデオの追加

2018年7月18日

ページ内エクスペリエンスに動画を追加するときに、再生リストの参照 ID を使用するオプションが追加されました。プレイリストの参照 ID を使用することで、Media モジュールを使用して参照 ID を別のプレイリストに再割り当てすることで、エクスペリエンスの動画を簡単に変更できます。 エクスペリエンスに表示されるビデオを変更する必要がある場合は、エクスペリエンスを編集して再公開する必要はありません。詳細については、プレイリストの参照 ID の使用を参照してください。

リードフォームへの変更

2018年5月15日

ギャラリーでは、オーディエンスモジュールを使用してリードフォームを作成および管理するようになりました。オーディエンスはリードデータを収集し、データのエクスポートオプションを提供します。詳細については、「ポータルエクスペリエンスでリードフォームを使用する」を参照してください。

ライブイベントのページ内テンプレート

2018年5月14日

ギャラリーでは、ページ内エクスペリエンスの一部としてライブイベントを配信するために使用できるライブイベントインページテンプレートが提供されるようになりました。詳細については、「ページ内エクスペリエンスのライブイベントを作成する」を参照してください。

ポータルエクスペリエンスサイト設定 UI の更新

2018年4月13日

サイト設定 UI が更新され、カスタムダイアログから新しいビデオ設定ダイアログにビデオ関連の設定が削除されました。詳細については、「ポータルエクスペリエンスのビデオ設定のカスタマイズ」を参照してください。

マーキーテンプレートの設定

2018年3月30日

マーキーテンプレートに、新しい注目のビデオ動作設定が追加されました。詳細については、「マーキーテンプレート設定のカスタマイズ」を参照してください。

カスタムヘッダーとフッターオプション

2018年3月30日

カスタムヘッダーとフッターを使用するときに、デフォルトのヘッダーとフッターを含めることができるようになりました。詳細については、「カスタムヘッダー/フッター HTML を構成する」を参照してください。

ページ内エクスペリエンス 3Play コンポーネント

2018年2月1日

3Play Media インタラクティブトランスクリプトコンポーネントをインページエクスペリエンスに追加できるようになりました。これにより、視聴者は動画の再生と同時に動画トランスクリプトを見ることができます。詳細については、「ページ内エクスペリエンスへのコンポーネントの追加」を参照してください。

ページ内エクスペリエンスチャットコンポーネント

2018年1月17日

チャットコンポーネントをページ内エクスペリエンスに追加できるようになりました。詳細については、「ページ内エクスペリエンスへのコンポーネントの追加」を参照してください。

カスタマイズされたアクセス制御ページ

2017年11月28日

アクセス制御ページは、サイトの他の部分とより密接にブランド化できるように、スタイルを設定できるようになりました。詳細については、「アクセス制御ページのスタイルを設定する」を参照してください。

定型化された 404 ページ

2017年11月6日

404 ページは、サイト (ヘッダーとフッター) で構成されたスタイルを使用するため、サイトの残りの部分とより密接にブランド化されています。

ライブイベントテンプレート-背景画像のオプション

2017年11月6日

ライブイベントテンプレートには、背景画像の処理方法に関するオプションが提供されるようになりました。詳細については、「ライブイベントポータルの設定のカスタマイズ」を参照してください。

コレクションメタデータ

2017年11月6日

すべてのテンプレート (マーキーを除く) で、コレクションの説明を追加でき、カテゴリページ (コレクションを共有するブラウザの場合) のメタデータの説明として機能します。詳細については、「ポータルエクスペリエンスへの Video Cloud ビデオの追加」を参照してください。

API キーは不要になりました

2017年9月29日

API キーは不要になり、[設定] ページから削除されました。

同じページに複数のインページエクスペリエンス

2017年8月15日

ギャラリーでは、同じページに同じページ内エクスペリエンスの複数のインスタンスを持つことがサポートされるようになりました。これにより、1 つのエクスペリエンスを作成し、表示するプレイリストまたはビデオのリストを動的に渡すことができます。詳細については、「ページ内エクスペリエンスでダイナミックビデオオーバーライドを使用する」を参照してください。

イベントエクスペリエンスが参照 ID の使用をサポートするようになりました

  2017年8月9日

イベントエクスペリエンスのライブイベント状態を設定するときに、参照 ID を使用してライブビデオアセットを指定できるようになりました。参照 ID を使用すると、エクスペリエンスに変更を加えることなく、参照 ID をライブビデオアセットに割り当てることができるため、柔軟性が向上します。詳細については、「イベントエクスペリエンスのライブイベント状態を作成する」を参照してください。

ページ内エクスペリエンスの追加プレイヤー設定

  2017年7月27日

新しいプレーヤー設定が追加され、ビデオの完了時にページ内エクスペリエンスが再生状態のままになります。これは、プレーヤーのエンドスクリーンを表示するために使用することができます。詳細については、「ページ内エクスペリエンスの設定を構成する」を参照してください。

ページ内エクスペリエンスのダイナミックビデオのオーバーライド

  2017年7月27日

埋め込みコードを変更することで、ページ内エクスペリエンスに関連付けられた動画を動的に上書きできるようになりました。詳細については、「ページ内エクスペリエンスでダイナミックビデオオーバーライドを使用する」を参照してください。

ページ内エクスペリエンストレーニングが利用可能になりました

  2017年7月21日

インページエクスペリエンスを作成するためのオンデマンドトレーニングが利用可能になりました。詳細については、「オンデマンドトレーニング:ギャラリーを使用してページ内エクスペリエンスを構築する。

YouTube 動画によるページ内エクスペリエンスのサポート

  2017年7月11日

ページ内エクスペリエンスで YouTube 動画を使用できるようになりました。詳細については、「ページ内エクスペリエンスに動画を追加する」を参照してください。

検索エンジンの結果でサイト説明を使用する

  2017年7月10日

Gallery では、検索エンジンの結果でサイトの説明を使用するオプションが提供されるようになりました。これは、Facebook や Twitter のメタ タグを含む <meta> の説明タグでサイトの説明を使用します。詳細については、「ポータル・エクスペリエンスの検索エンジン最適化設定の構成」を参照してください。

カタログテンプレート

2017年6月28日

ギャラリーでカタログテンプレートが提供されるようになりました。詳細については、「ギャラリーポータルテンプレートの概要」を参照してください。

ページ内エクスペリエンス

2017年5月31日

ギャラリーでは、ページ内エクスペリエンスを作成するオプションが提供されるようになりました。詳細については、「概要:ページ内エクスペリエンスの作成、編集、公開。
