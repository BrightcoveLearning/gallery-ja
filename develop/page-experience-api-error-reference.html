ページ内エクスペリエンス API エラーリファレンス

このトピックでは、ページ内エクスペリエンス API (プラットフォーム API) のエラーリファレンスを提供します。

ページ内エクスペリエンス API エラー

ページ内エクスペリエンス API エラーリファレンス
エラー 例 説明
400 BAD_REQUEST 
{
  "status": 400,
  "code": "BAD_REQUEST",
  "message": "No template specified"
}
{
  "status": 400,
  "code": "BAD_REQUEST",
  "message": "Specified template does not exist"
}
{
  "status": 400,
  "code": "BAD_REQUEST",
  "details": [
    "Path `name` is required."
  ]
}
 通常、リクエスト本文に問題がある。たとえば、templateまたはその他の必須フィールドを省略したり、存在しないテンプレートを指定したりする
401 UNAUTHORIZED 
[
  {
    "error_code": "UNAUTHORIZED",
    "message": "Permission denied."
  }
]
 リクエストでアクセストークンが送信されなかったか、アクセストークンの作成に使用されたクライアント資格情報に、要求に対する正しいアクセス許可がありませんでした
404 NOT_FOUND 
{
  "status": 404,
  "code": "NOT_FOUND",
  "message": "Experience 5bc2624e0f6f340011b not found"
}
 更新しようとしているエクスペリエンスが存在しません。削除されたか、リクエスト内のエクスペリエンス ID が正しくない可能性があります。
