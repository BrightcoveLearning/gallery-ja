ギャラリーエクスペリエンスの概要

{{ page.description }}

Brightcove Galleryを使用すると、カスタム開発を必要とせずに、強力で魅力的なビデオエクスペリエンスを簡単に作成できます。ギャラリーでは、次のタイプのエクスペリエンスを作成できます。

  • ポータル
  • ページ内
  • ランディングページ
  • イベント
注:Gallery エクスペリエンスの作成と公開をすばやく開始するには、ポータルエクスペリエンスまたはページ内エクスペリエンスの作成に関するステップバイステップのトピックを参照してください。

作成するエクスペリエンスのタイプを選択すると、Gallery には、ポータル（ランディングページとイベントエクスペリエンスを含む）とページ内エクスペリエンスの両方に対して、エクスペリエンスの全体的なルックアンドフィール。

ポータル

ポータルエクスペリエンスは、ブライトコーブがホストする指定された URL でアクセスされるスタンドアロンの動画サイト（動画ポータル）です。次の例では、Video Cloud および Gallery トレーニングビデオが Portal Experience に編成されています。左側のカテゴリを選択すると、ページの本文にそのカテゴリに関連付けられているビデオが表示されます。

ページ内

ページ内エクスペリエンスは、既存の Web ページ内に埋め込まれます。ページ内エクスペリエンスを使用すると、動画レイアウトを作成し、既存のウェブサイト内に表示できます。インページエクスペリエンスは、動画再生時に表示できるリンクやカードなど、プレイヤー内およびプレイヤー外の両方のインタラクティブ性をサポートします。次の例では、ページ内エクスペリエンス（動画のサムネイル）が既存の Web ページに埋め込まれています。ビデオサムネールの 1 つをクリックすると、ライトボックスにビデオプレーヤーが開きます。

ランディングページ

ランディングページエクスペリエンスは、ブライトコーブがホストする単一の動画ページで、通常はメールキャンペーンの一部として使用されます。ユーザーが電子メールメッセージ内のリンクをクリックすると、ランディングページエクスペリエンスにリダイレクトされ、単一のビデオを表示できます。以下の例には、視聴者情報を取得するために使用できるリードフォームも含まれています。

イベント

イベントエクスペリエンスでは、ブランド化されたイベント前の目的地、ライブストリームイベントページ、イベント後の動画アーカイブを作成して、ライブイベント前、中、後にオーディエンスをエンゲージすることができます。イベントエクスペリエンスは、スタンドアロン、ポータルサイトとして作成することも、ページ内エクスペリエンスとして既存のページに埋め込むこともできます。次の例では、ライブイベントポータルが作成され、イベント前の状態を表示するように構成されています（ホームページのカウントダウンタイマーに注意してください）。
