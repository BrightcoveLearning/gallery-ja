ギャラリーエクスペリエンスの作成と管理

{{ page.description }}

ギャラリーエクスペリエンスの作成と公開をすばやく開始する場合は、次のいずれかのドキュメントを参照してください。

  • ステップバイステップ:ポータルエクスペリエンスの作成と公開
  • ステップバイステップ:ページ内エクスペリエンスの作成と公開

ギャラリーモジュールを開くと、エクスペリエンスページに、作成されたすべてのエクスペリエンスが一覧表示されます。

エクスペリエンスが作成されていない場合は、エクスペリエンスを作成するように求められます。エクスペリエンスページでは、次の機能が提供されます。

  • エクスペリエンスの削除
  • 体験をオフラインにする
  • 重複したエクスペリエンス
  • エクスペリエンスの編集
  • エクスペリエンスを作成する
  • エクスペリエンスのフィルタリング
  • エクスペリエンスの検索

データを並べ替えるには、列ヘッダーをクリックします。列セレクター（）を使用して、ページに表示する列を選択できます。

次の列が表示されることがあります。

  • タイトル -エクスペリエンスのタイトル
  • Template -エクスペリエンスで使用されるテンプレート。エクスペリエンスの種類は以下に表示されます。
    • ポータル -任意のサイズのコンテンツライブラリをサポートするサイト（オプション）アクションと広告サポートを提供
    • ページ内 -1 つ以上の動画を公開してインタラクティブ機能を追加してから、このレイアウトをサイトに追加します。
    • ランディングページ -リードフォームとアクションを促すオプションと広告サポートを備えた単一ページサイト
    • Event -ライブイベントの宣伝と放送に最適化されたサイト
  • テンプレート・ステータス -テンプレートのステータス
    • 最新版 -エクスペリエンスはプロジェクトテンプレートの最新バージョンを使用しています
    • 更新が必要 -エクスペリエンスのテンプレート更新が利用可能です
    • 更新の再開 -テンプレートの更新は進行中ですが、公開されていません
  • 更新済み -エクスペリエンスの最終更新日時
  • エクスペリエンスのステータス -エクスペリエンスのステータス
    • オンライン -エクスペリエンスはオンラインであり、アクセスすることができます
    • オフライン -エクスペリエンスはオフラインであり、アクセスできません
    • 未公開の変更点 -エクスペリエンスはオンラインであり、アクセスできますが、エクスペリエンスに変更が加えられたため、公開されていません

行をクリックすると、ページの右側にエクスペリエンスの詳細が表示されます。

エクスペリエンスの作成

新しいエクスペリエンスを作成するには、[ 新しいエクスペリエンス] をクリックします。新しいエクスペリエンスを作成するときは、まずテンプレートを選択します。テンプレートは、全体的な外観と動作を制御します。テンプレートを選択したら、エクスペリエンスをカスタマイズできます。詳細については、次のいずれかのトピックを参照してください。

  • 概要:ポータルエクスペリエンスの作成、編集、公開
  • 概要:ページ内エクスペリエンスの作成、編集、公開

エクスペリエンスの複製

エクスペリエンスの複製は、複数の言語でエクスペリエンスを作成する必要がある場合、または既存のエクスペリエンスのルックアンドフィールからマイクロサイトを作成する場合に便利です。エクスペリエンスを複製するには、行をクリックし、[ 複製] をクリックします。

エクスペリエンスのタイトルと説明を入力し、[ 保存] をクリックします。

エクスペリエンスの編集

エクスペリエンスを編集するには、エクスペリエンスのタイトルをクリックします。エクスペリエンスを編集すると、ギャラリーサイトエディタが開きます。エクスペリエンスの編集については、次のいずれかのトピックを参照してください。

  • サイトエディターを使用したポータルエクスペリエンスのカスタマイズ
  • ページ内エクスペリエンスのカスタマイズ

体験をオフラインにする

エクスペリエンスをオフラインにすると、アクセスできなくなります。エクスペリエンスをオフラインにするには、行をクリックし、[ オフラインにする] をクリックします。エクスペリエンスをオフラインにする必要があることを確認するには、[ オフラインにする] をクリックします。エクスペリエンスを再び利用できるようにするには、エクスペリエンスを再公開する必要があります。

エクスペリエンスの削除

エクスペリエンスを削除するには、行をクリックし、[削除] をクリックします。[削除]をクリックして、削除を確認します。

注:エクスペリエンスがカスタムドメインを使用するように構成されている場合は、エクスペリエンスに割り当てられたカスタムドメインを削除してから、エクスペリエンスを削除する前にエクスペリエンスを再公開する必要があります。

エクスペリエンスのフィルタリング

[エクスペリエンス] ページにフィルターを適用して、表示されるエクスペリエンスのタイプを制限できます。フィルターを適用するには、表示するエクスペリエンスのタイプを選択します。

  • すべて -すべてのエクスペリエンスを表示
  • 公開 -公開済みの (オンライン) 体験をすべて表示
  • 未公開 -オフラインになったすべてのエクスペリエンスを表示します
  • テンプレートの更新が必要 -テンプレートの更新が使用可能なすべてのエクスペリエンスを表示します
  • ランディングページ -すべてのランディングページエクスペリエンスを表示
  • イベント -すべてのイベント体験を表示
  • ポータル -ポータルエクスペリエンスを表示
  • ページ内 -すべてのページ内エクスペリエンスを表示

エクスペリエンスの検索

検索ボックスは、エクスペリエンスのリストを検索するために使用できます。検索時には、次のプロパティが検索されます。

  • タイトル
  • 説明
  • ID
  • テンプレート
  • URL
