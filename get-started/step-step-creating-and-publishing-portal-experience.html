    ステップバイステップ:ポータル・エクスペリエンスの作成と公開

    このトピックの目的は、ブライトコーブギャラリーを使用して魅力的なポータルエクスペリエンスを作成する方法を説明することです。
     

    このトピックを完了すると、次のことができるようになります。

    • Gallery設定を行う
    • ギャラリーを使用して新しいポータルエクスペリエンスを作成する
    • ポータルエクスペリエンスに動画を追加する
    • ポータルエクスペリエンスのカスタマイズ
    • ポータルエクスペリエンスのプレビューと公開
    注：ページ内エクスペリエンスの作成に関するステップバイステップガイドについては、「ステップバイステップ:ページ内エクスペリエンスの作成と公開。

    Audience

    カスタム開発を必要とせずに、魅力的なポータルエクスペリエンスを作成する必要がある Video Cloud パブリッシャーです。

    前提条件

    • ギャラリーモジュールにアクセスできる Video Cloud アカウント
    • Video Cloudについての実用的な知識
    • このトピックで使用されているサンプルビデオは、必要に応じてダウンロードできます
    • ギャラリートレーニングを使用して Building Portal エクスペリエンスを表示/参加するのに役立つ

    始める前に

    提供されているサンプルビデオを使用するには、次の手順を実行します。

    1. サンプルファイルをダウンロードします。
    2. Video Cloudアカウントに動画をアップロードします。
    3. チッカディーの動画にbird in flight「」のタグを追加します。
    4. 3 つのプレイリストを作成し、次のように再生リストにビデオを追加します。
      動画 プレイリスト

      Crab
      Dolphins
      Leafy Seadragon
      Lionfish
      Sea Turtle
      Seahorse      		 Sea Creatures
      Chickadee
      Peacock
      Woodpecker      		 Birds
      Elephant
      Tiger
      Zebra      		 Animals
    5. いくつかの動画に関連リンク URL を追加します。後のセクションでは、ビデオの関連リンクを表示するようにポータルを構成します。
    6. プレイヤーモジュールを使用して、ギャラリープレーヤーという新しいプレーヤーを作成します。
    7. タイトルと説明を表示しないようにプレーヤーを設定します。プレーヤーを公開します。

    ポータルエクスペリエンスの作成

    ブライトコーブギャラリーを使用してポータルエクスペリエンスを作成するには、以下のタスクを実行します。

    • 既定のプレーヤーでギャラリーの構成
    • カスタムドメイン名の作成
    • 新しいポータルエクスペリエンスの作成
    • サイトの詳細を編集する
    • ページテンプレートへのオブジェクトの追加
    • エクスペリエンスをプレビューする
    • コレクションを作成し、コレクションにビデオを追加する
    • サイトスタイルの変更
    • サイト検索を有効にする
    • ソーシャル設定をカスタマイズする
    • ビデオと再生の設定をカスタマイズする
    • カスタムドメインを割り当てる
    • エクスペリエンスを公開する

    デフォルトのプレーヤーでギャラリーの構成

    ブライトコーブギャラリーは、ギャラリーサイトで使用されるデフォルトの動画プレーヤーで設定できます。デフォルトプレーヤーが指定されていない場合、 Brightcove のデフォルトプレーヤーがデフォルトになります。既定のプレーヤーで Gallery を構成するには、次の手順を実行します。

    1. ビデオクラウドスタジオにログインします。
    2. お客様の現在のVideo Cloudアカウント名がページの右上隅に表示されます。複数のアカウントがある場合は、アカウントセレクタのドロップダウンをクリックし、エクスペリエンスを作成するアカウントを選択します。このアカウントには、使用したい動画が存在している必要があります。
    3. ナビゲーションメニューの [ ギャラリー ] をクリックします。
    4. ページ上部の [ 設定 ] リンクをクリックします。
    5. お客様のGalleryサイトで使用されるプレーヤーを選択します。手順を実行する場合は、前に作成したギャラリープレーヤーを選択します。
      注：ベストプラクティスとして、ギャラリー体験のために専用のプレーヤーを作成してスタイルを設定することをお勧めします。
    6. [保存]をクリックします。

    カスタムドメイン名の作成

    新しいポータルエクスペリエンスを作成すると、ギャラリーはランダムな URL を割り当てます（例site-12345.bcvp0rtal.com）。ポータルエクスペリエンスには、ブライトコーブが提供するドメイン（このトピックで説明します）を割り当てることも、独自のカスタムドメインを使用することもできます。ブライトコーブが提供する新しいドメインを作成するには、次の手順に従います。

    1. [ 設定 ] ページで、左側のナビゲーションで [ カスタムドメインと SSL ] をクリックします。
    2. [カスタムドメインの作成]をクリックします。
    3. brightcovegallery.com ドメインまたは gallery.video ドメインのいずれかを選択します。
    4. カスタムドメインを入力します。自分の名前または会社名を使用してください。ドメインが使用できない場合は、メッセージが表示されます。別の名前を入力します。
    5. 有効なドメインを入力したら、[ 保存 ] をクリックします。
    注：カスタムドメインは、後のセクションでポータルに割り当てられます。

    新しいポータル・エクスペリエンスの作成

    新しいギャラリーポータルエクスペリエンスを作成するには、次の手順に従います。

    1. ページ上部の [ エクスペリエンス ] リンクをクリックします。
    2. [ 新しいエクスペリエンス ] をクリックします。
    3. ページ上部の [ ポータル ] タブをクリックします。ポータル・テンプレートのリストが表示されます。マーキーテンプレートを見つけて、[ 選択 ] をクリックします。
      注：テンプレートを選択すると、そのテンプレートは変更できません。ギャラリーテンプレートの詳細については、「ギャラリーポータルテンプレートの概要」トピックを参照してください。
    4. エクスペリエンスのタイトルに「 Animal World 」と入力します。事前入力エクスペリエンスのために？設定で、[ いいえ ] を選択します。
    5. [ エクスペリエンスの作成 ] をクリックします。

    Gallery Site Editorが開きます。ギャラリーサイトエディタは、エクスペリエンスをカスタマイズするために使用され、ページレイアウトの管理、ビデオの追加、スタイルの選択、ポータルのその他の設定の構成を行うことができます。すべてのカスタマイズ設定は、左側のナビゲーションメニューを使用してアクセスします。

    サイトの詳細を編集する

    エクスペリエンスのタイトル、説明、言語設定は、ギャラリーサイトエディタを使用して編集できます。

    1. 左側のナビゲーションで [ OVERVIEW ] をクリックします。
    2. エクスペリエンスの説明に、アニマルワールドビデオポータルを追加します。
    3. [保存]をクリックします。

    ページテンプレートにオブジェクトを追加する

    各ポータル・テンプレートには、カスタマイズ可能なページ・テンプレートのセットが用意されています。提供されるページテンプレートは、エクスペリエンスの初回作成時に選択されたテンプレートによって異なります。マーキーテンプレートには、ホーム、再生、およびカテゴリページのテンプレートが用意されています。これらのページは、ページメニューの一部として左余白に表示されます。

    ページ・テンプレートへのオブジェクトの追加の詳細は、「ポータル・エクスペリエンスページのコンテンツのカスタマイズ」を参照してください。

    ページテンプレートにオブジェクトを追加するには、次の手順を行います。

    1. 左側のナビゲーションで [ PAGES ] をクリックします。
    2. 左マージンの[Home]ページをクリックします。プラスアイコン（）は、オブジェクトを追加できるページ上の領域を示します。ページ上の静的テキストオブジェクトは、クリックして編集できます。
    3. ページ上部の [ + 画像またはテキストを追加 ] をクリックします。
    4. [ イメージ ] タブをクリックします。
    5. [ 参照 ] をクリックし、[ アニマルワールド Logo.png ] を参照して選択します。このファイルはサンプルアセットの一部として提供されています。[保存]をクリックします。
    6. 画像がページヘッダーに表示されることを確認します。
    7. ページの一番下までスクロールします。[ あなたのブランド] をクリックします。タイトルに Animal World と入力し、[Save]をクリックします。
    8. 左余白の [ 再生 ] ページと [ カテゴリ ] ページをクリックします。ヘッダーイメージとタイトルテキストがそれらのページにも表示されることに注意してください。[Home]ページに戻ります。

    エクスペリエンスのプレビュー

    エクスペリエンスをプレビューすると、モバイル、タブレット、デスクトップデバイスでポータルがどのように表示されるかを確認できます。ポータルをカスタマイズする手順に従って、ポータルをプレビューする必要があります。ポータルを新しいブラウザータブでプレビューしたり、共有可能なリンクを生成して他のユーザーがポータルをプレビューしたりできます。ポータルエクスペリエンスのプレビューと公開の詳細については、「ポータルエクスペリエンスのプレビューと公開」を参照してください。

    ポータルをプレビューするには、次の手順に従います。

    1. ギャラリーサイトエディタを使用してサイトを編集しているときに、[ プレビュー ] をクリックし、[ 応答プレビュー ] をクリックします。
    2. プレビューが新しいブラウザタブで開きます。
    3. [サイトのプレビュー] タブを閉じます。

    コレクションを作成し、コレクションに動画を追加する

    ギャラリーサイトエディタを使用して、ポータルに表示するビデオを選択できます。ポータルエクスペリエンスを作成すると、ビデオはコレクションに編成されます。コレクションは Video Cloud 動画メタデータに基づいて作成され、以下を含めることができます。

    • Video Cloudプレイリスト
    • キーワード
    • カスタムメタデータ
    • タグ

    YouTubeアカウントの動画をコレクションに追加することもできます。コレクションの作成およびサイトへのビデオの追加の詳細については、「ポータルエクスペリエンスへのビデオの追加」を参照してください。

    ポータルエクスペリエンスにビデオを追加するには、次の手順に従います。

    1. 左側のナビゲーションで [ 動画 ] をクリックします。
    2. 最初のコレクションに Birds という名前を付けて、Enter キーを押します。
    3. コレクション名をクリックして、コレクションエディタを開きます。
    4. コレクションソースとして [ ビデオクラウド ] を選択し、条件として [ プレイリスト ] を選択します。
    5. [ 鳥 ] プレイリストをクリックします。バードプレイリスト内のすべての動画が動画リストに表示されます。[保存]をクリックします。
      注：再生リストのビデオ割り当てを使用する場合、ビデオの順序は、Media モジュールで設定された再生リスト設定に基づきます。
    6. Animals プレイリストを条件として使用して、Animals という名前の別のコレクションを作成します。
    7. 海の生き物のプレイリストを条件として使用して、海の生き物という名前の別のコレクションを作成します。
    8. フライト中の鳥という名前の別のコレクションを作成します。基準として [ カスタム ] を選択します。
    9. 動画に含めるには、[ タグ ] を選択し、' bird in flight ' のタグを追加します。Enterキーを押します。
    10. 最新順でビデオを注文します。
    11. Chickadeeビデオがビデオコレクションリストに表示されていることを確認します。[保存]をクリックします。
    12. 4 つのコレクションが作成されていることを確認します。
      注：コレクションをクリックして新しい位置にドラッグすることで、コレクションの順序を変更できます。

    サイトスタイルの変更

    スタイル設定は、エクスペリエンスで使用されるサイトの色とフォントをカスタマイズするために使用できます。サイトスタイルの変更の詳細については、「ポータルエクスペリエンスのスタイルのカスタマイズ」を参照してください。

    サイトスタイルを変更するには、次の手順に従います。

    1. 左側のナビゲーションで [ スタイル ] をクリックします。
    2. テーマは、テンプレートに適用できる色とフォントのセットです。テーマを選択します。このポータルでは、 Triton テーマを使用します。
    3. 使用するPrimaryおよびSecondaryフォントを選択します。このポータルでは、 Open Sans フォントを使用します。
    4. スタイルメニューの上部にある [ < エクスペリエンスに戻る ] リンクをクリックします。

    外観と動作の設定の一部として、その他のスタイル関連の設定も使用できます。

    1. 左側のナビゲーションで [ > 外観と動作テンプレートの設定] をクリックします。
    2. [ 注目の動画を選択したカテゴリの最初の動画に変更する ] を選択します。
      注：使用できるテンプレート設定は、エクスペリエンスの作成時に選択したテンプレートに基づいています。すべてのテンプレートにテンプレート設定があるわけではありません。
    3. [保存]をクリックします。

    外観と動作の設定の詳細については、「ポータルエクスペリエンスの外観と動作のカスタマイズ」を参照してください。

    サイト検索の有効化

    サイト検索の設定を使用して、サイト検索機能を有効化およびカスタマイズできます。サイト検索を有効にすると、検索機能がサイトページに追加され、視聴者はお客様のコンテンツを検索できます。サイト検索のカスタマイズの詳細については、「ポータルエクスペリエンスのサイト機能のカスタマイズ」を参照してください。

    サイト検索を有効にするには、次の手順に従います。

    1. 左側のナビゲーションで [ > サイト機能] [サイト検索 ] をクリックします。
    2. [ サイト検索を有効にする ] オプションを選択します。
    3. [ 保存 ] をクリックして、設定を保存します。

    ソーシャル設定のカスタマイズ

    [Social] 設定では、視聴者は人気のあるソーシャルメディアサイトにコンテンツを簡単に共有できます。また、視聴者がお客様のソーシャルメディアサイトに迅速にアクセスする方法も提供できます。ソーシャル設定のカスタマイズの詳細については、「ポータルエクスペリエンスのソーシャル設定のカスタマイズ」を参照してください。

    ソーシャル設定を構成するには、次の手順を実行します。

    1. 左側のナビゲーションで [ サイト機能] [> ソーシャル ] をクリックします。
    2. [Sharing Options]をすべて選択します。これにより、プレーヤーの横にソーシャル共有アイコンが追加されます。
    3. [ ライトに暗いアイコン ] オプションを選択します。
    4. [保存]をクリックします。
    5. ページ上部の [ 接続状態を維持する ] タブをクリックします。お客様のソーシャルメディアアカウント情報を提供すると、Galleryサイトではページのフッターにアイコンが表示され、視聴者はお客様のソーシャルメディアサイトに直接アクセスできます。
    6. ソーシャルメディア情報を入力するか、「Facebook名」、「Twitterハンドル」、「LinkedInユーザー名」に「brightcove」と入力します。
    7. [ 暗い時に明るいアイコン] オプションを選択します。
    8. [ 保存 ] をクリックして、設定を保存します。

    ビデオと再生の設定をカスタマイズする

    ビデオと再生の設定を使用して、エクスペリエンスのカスタムプレーヤーの選択、リードフォームの追加、その他のビデオ関連設定のカスタマイズを行うことができます。ソーシャル設定のカスタマイズの詳細については、「ポータルエクスペリエンスのビデオと再生設定のカスタマイズ」を参照してください。

    ビデオと再生の設定をカスタマイズするには、次の手順を実行します。

    1. ギャラリーサイトエディタで、[ > ビデオと再生プレーヤー] と [リードフォーム ] をクリックします。
    2. [ プレーヤーの動作 ] の設定が両方ともオンになっていることを確認します。必要に応じて [ 保存 ] をクリックします。
    3. 左側のナビゲーションで [ > ビデオと再生 ] [ビデオ] をクリックします。
    4. [ ビデオごとに関連リンクを表示 ] オプションを選択します。
    5. [保存]をクリックします。

    カスタムドメインの割り当て

    ポータルエクスペリエンスが作成されると、それらにはランダムな静的 URL が割り当てられます。ほとんどの場合、より分かりやすい URL でサイトにアクセスする必要があります。ポータル・エクスペリエンスへのカスタム・ドメインの割り当ての詳細については、「ポータル・エクスペリエンスへのカスタム・ドメインの割り当て」を参照してください。

    ポータルに割り当てられた URL を表示し、カスタム URL を割り当てるには、次の手順に従います。

    1. ギャラリーサイトエディタで、[ サイト構成 > URL ] をクリックします。サイトに割り当てられた URL が表示されます。
    2. [カスタムドメインの追加]をクリックします。
      注：[ カスタムドメインの追加 ] ボタンがない場合は、カスタムドメインは作成されていません。
    3. 前に作成したドメインを選択します。
    4. [保存]をクリックします。

    エクスペリエンスの公開

    ポータルエクスペリエンスを公開すると、パブリック URL でエクスペリエンスが利用可能になります。ポータルエクスペリエンスのプレビューと公開の詳細については、「ポータルエクスペリエンスのプレビューと公開」を参照してください。

    エクスペリエンスを公開するには、次の手順に従います。

    1. [ギャラリーサイトエディタ] で、[ 発行 ] をクリックします。
      注：サイトに割り当てたカスタムドメインがダイアログに表示されます。
    2. [サイトをパブリッシュ] ダイアログで、[ 今すぐ発行 ] をクリックして、サイトの公開を確認します。
